***To integrate with Zoho Recruit follow, the below steps.***


## Process of Authorization and Integration
> **Note: Recruit is only available on the pro version.**

1.	Search & Select **Zoho Recruit** from the actions pages
 <Image src="actions/zoho-recruit/_select-zoho-recruit" alt="Zoho Recruit" w="1020" h="350" />

4.	Integration Name: Set the proper integration name Default it's taken “Zoho Recruit”.
 <Image src="actions/zoho-recruit/_integration-name" alt="Zoho Recruit" w="1020" h="350" />

5. Select a data center from the dropdown. Then click on the *Zoho API Console*. Navigate to **API Console-**> **Get Started**-> **Server-based Applications.** 
 <Image src="actions/zoho-recruit/_select-data-center" alt="Zoho Recruit" w="1020" h="350" />
 <Image src="actions/zoho-recruit/_get-started" alt="Zoho Recruit" w="1020" h="350" />
 <Image src="actions/zoho-recruit/_server-based-applications" alt="Zoho Recruit" w="1020" h="350" />


6.	Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click **CREATE** button
 <Image src="actions/zoho-recruit/_create-api" alt="Zoho Recruit" w="1020" h="350" />

 
7.	A Client ID and a Client Secret will be generated. Then go to the Client Secret tab
 <Image src="actions/zoho-recruit/_client-id-and-secret" alt="Zoho Recruit" w="1020" h="300" />

8.	Then copy the Client ID and Client Secret and paste it to the Bit Integrations integration page. Then click on the **Authorize** button (wait for a few seconds). After the successful authorization click on the **Next** button
 <Image src="actions/zoho-recruit/_authorization" alt="Zoho Recruit" w="1020" h="300" />

9.	Select a **Module** from the drop-down list. Map all the mandatory Recruit fields to the corresponding fields in your form.
 <Image src="actions/zoho-recruit/_select-module" alt="Zoho Recruit" w="1020" h="300" />
 <Image src="actions/zoho-recruit/_field-mapping" alt="Zoho Recruit" w="1020" h="300" />

10. If you use File Upload field should to map this separately.
 <Image src="actions/zoho-recruit/_attachments" alt="Zoho Recruit" w="1020" h="300" />
 
## Related List

 1. You can send data to Zoho Recruit related list from Trigger. Select the required related list module and layout from the drop-down provided. Then map the mandatory field. To map more fields, click on the (+).
   <Image src="actions/zoho-recruit/_related-list" alt="Zoho Recruit" w="1020" h="350" />

## Actions

1. **Workflow:** Select this option to trigger the Zoho Recruit workflow for the selected module. If you didn’t select this option Zoho Recruit workflow for the selected module doesn’t work.
   <Image src="actions/zoho-recruit/_workflow-actions" alt="Zoho Recruit" w="1020" h="350" />

2. **Approval:** Select this option to automatically send form data to your Zoho Recruit approval list instead of adding them.
    <Image src="actions/zoho-recruit/_approval-actions" alt="Zoho Recruit" w="1020" h="350" />

3. **Upsert Record:** : When you choose the Upsert Record option, if a record with the identical value exists in Zoho Recruit, then it will be updated with the new values. If not, a new record will be created.
   <Image src="actions/zoho-recruit/_upsert-record-actions" alt="Zoho Recruit" w="1020" h="350" />

4. **Add Note:** Select this action to add a note from Trigger to Zoho Recruit account automatically.
   <Image src="actions/zoho-recruit/_add-notes-actions" alt="Zoho Recruit" w="1020" h="350" />

5. **Record Owner:** YSelect this option you can set a record owner of the sent record. For this, you have to copy the Owner ID from Zoho Recruit and paste it here.
   <Image src="actions/zoho-recruit/_record-owner-actions" alt="Zoho Recruit" w="1020" h="350" />

## Save Integration
  1. After giving all the required information, click on **Finish & Save**. Now your created **Zoho Recruit API ** is ready for implementation 
   <Image src="actions/zoho-recruit/_finish-and-save" alt="Zoho Recruit" w="1020" h="350" />
