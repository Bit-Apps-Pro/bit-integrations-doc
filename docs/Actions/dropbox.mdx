***To integrate with Dropbox follow, the below steps.***


## Process of Authorization and Integration


1.	Search & Select **Dropbox** from the actions pages
 <Image src="actions/dropbox/search-and-select-dropbox" alt="Dropbox" w="1020" h="350" />

2.	**Integration Name:** Set the proper integration name. Default it's taken ‘Dropbox Integration’.
 <Image src="actions/dropbox/dropbox-integration-name" alt="Dropbox" w="1020" h="350" />

3.	After that, To get the API key & API Secret, click on `Dropbox API Console`.
    1. **Choose an API:**  Scoped access
    2. **Choose the type of access you need:**  Full Dropbox
    3. Type Name your app and click on **`Create App`** button. 
    4. **Go to Permissions Tab:** **Permissions that allow your app to view and manage** > you need to enable all scopes. <br/>Now copy the API key and API secret, then paste it into the integration’s dashboard.
     <Video src="dropbox-api" alt="Dropbox" w="1020" h="350" />
      
    2. After that, to get the Access Code, click on **Dropbox Access Code**. And Create a Access Code. Now copy the Generated Access Code and paste it into the integration’s dashboard. And Click on **`“Authorize”`** button.
      <Video src="dropbox-access-code" alt="Dropbox" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.
 <Image src="actions/dropbox/dropbox-authorize-after-next" alt="Dropbox" w="1020" h="350" />

5.	You will find the Map Field option. Select the `folder Name` from the Drop-Down list that you want to send the files to the folder.
 <Image src="actions/dropbox/dropbox-folder-select" alt="Dropbox" w="1020" h="350" />

## Actions

6.	Also, if you want to delete the uploaded file from WordPress, enable the `'Delete File From WordPress'` option.

 <Image src="actions/dropbox/delete-file-form-wordpress" alt="Dropbox" w="1020" h="350" />

## Save Integration

7.	Now click on `Next` button. And save the integration by clicking the `Finish & Save` Button.
 <Image src="actions/dropbox/integrations-finish-and-save" alt="Dropbox" w="1020" h="350" />

8.	We have an option in every integration `Timeline` where you can check integration success or error status.
 <Image src="actions/dropbox/integrations-timeline-success-and-error" alt="Dropbox" w="1020" h="350" />
 <Image src="actions/dropbox/timeline-success-and-error" alt="Dropbox" w="1020" h="350" />

9.	Also, you can `Edit` `Clone` and `Delete` the integration.
 <Image src="actions/dropbox/integrations-edit-clone-delete" alt="Dropbox" w="1020" h="350" />
