***To integrate with Encharge follow, the below steps.***


## Process of Authorization and Integration


1.	Search & Select **Elastic Email** from the actions pages
 <Image src="/actions/elastic-mail/select-action" alt="Elastic Email" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken Elastic Email.
 <Image src="/actions/elastic-mail/integration-name" alt="Elastic Email" w="1020" h="350" />

3.	After that, To get the API key, click on **Elastic Email API Console** or **Login your Elastic Email dashboard** => **click on My Account** => **Setting** => **Create Additional API Key** . You need to grant `Full Access` **permission** to make the API successful. Now copy the API key and paste it into the integrations dashboard. And Click on **“Authorize”** button

 <Image src="/actions/elastic-mail/create-api" alt="Elastic Email" w="1020" h="350" />
 <Image src="/actions/elastic-mail/copy-api" alt="Elastic Email" w="1020" h="350" />
 <Image src="/actions/elastic-mail/authorize" alt="Elastic Email" w="1020" h="350" />


4. Click on Next to set up the other integration settings ahead.
 
5. After completing the authorization process, Select the List from the drop-down option. You have to create a list before on your Elastic Email account. 

 <Image src="/actions/elastic-mail/elastic-mail-list" alt="Elastic Email" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary form fields with the related Elastic Email field. Here **‘Email‘** field is the default for Elastic Email. Now, click on the Next button to move forward for further settings.
 <Image src="/actions/elastic-mail/elastic-mail-field-map" alt="Elastic Email" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="/actions/elastic-mail/finish-save" alt="Elastic Email" w="1020" h="350" />

8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="/actions/elastic-mail/elastic-mail-timeline" alt="Elastic Email" w="1020" h="350" />
 <Image src="/actions/elastic-mail/elastic-mail-log" alt="Elastic Email" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="/actions/elastic-mail/elastic-mail-option" alt="Elastic Email" w="1020" h="350" />
