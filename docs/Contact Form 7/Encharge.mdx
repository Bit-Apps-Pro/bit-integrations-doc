***To integrate with Encharge follow, the below steps.***


## Process of Authorization and Integration
1.	At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **“Create Integrations”** button.
 <Image src="Encharge/Create Integration" alt="Encharge" w="1020" h="350" />


2. Now select Contact Form 7 that you want to integrate with Encharge.
 <Image src="Encharge/Select Contact Form 7" alt="Encharge" w="1020" h="350" />

3. Select the form name from the Drop-Down list that you want to integrate with Encharge. And click on the **“Next” button.** And select Encharge.
 <Image src="Encharge/Select Form" alt="Encharge" w="1020" h="350" />
 <Image src="Encharge/Encgarge" alt="Encharge" w="1020" h="350" />

4. Integration Name: Set the proper integration name. Default it's taken “Encharge”.
 <Image src="Encharge/Integration Name" alt="Encharge" w="1020" h="350" />

5.	After that, you have to copy the API key from your Encharge account & paste the key on the field. To get the API key, click on **Encharge CONSOL**=> **click on Account icon** => Your account. And Click on **“Authorize”** button
 <Image src="Encharge/Encgarge API Console" alt="Encharge" w="1020" h="350" />
 <Image src="Encharge/API Console Dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="Encharge/Authorize" alt="Encharge" w="1020" h="350" />


6. Click on Next to set up the other integration settings ahead.
 
7. After completing the authorization process, you will find the Map Field option. You have to map all the necessary fields with the related Encharge field. By default “Email” field of Encharge will be available. You can also send custom values by mapping your required field
 <Image src="Encharge/Field Mapping" alt="Encharge" w="1020" h="350" />

9.	We have an option in every integration "Timeline," where you can check integration success or error status.
 <Image src="Encharge/Integrations Dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="Encharge/Timeline" alt="Encharge" w="1020" h="350" />
