In this section we will show you how you can install Bit Integrations on your WordPress site & also how you can activate Bit Integrations pro version.

## Process of installation & activation

Follow the path mentioned below:

1. At first, navigate to your wordpress & navigate to plugin. Then click on `Add new`.
2. Go to search option & search ***Bit Integrations***.

  <Image src="install/free-bit-integrations-plugin-search" alt="plugin install wordpress" w="758" h="270"/>

  <Image src="install/search-and-install-bit-integrations-plugin" alt="plugin install wordpress" w="698" h="415"/>

3.  Click `Install now` & then `Activate` free version.
  <Image src="install/active-bit-integrations-free-version" alt="plugin install wordpress" w="1046" h="200"/>


4.  Now you can see **Bit Integrations** in your WordPress plugin list.

  <Image src="install/view-bit-integraions-free-version-plugin" alt="plugin install wordpress" w="1047" h="200"/>


5.  After successful installation you can see the home page of **Bit Integrations**'. Now You are ready to create Integration.

  <Image src="install/bit-integrations-dashboard" alt="plugin install wordpress" w="1114" h="350"/>

## Process of activating premium version 

1. At first go to our website [bitapps.pro/bit-integrations](https://bitapps.pro/bit-integrations)
2. Go to **Pricing** section of pro version.

 <Image src="install/pricing" alt="plugin install wordpress" w="1114" h="350"/>


3. Now you can see all the plans of Bit Integrations.
4. Select your desired plan & you can customize the quantity of sites.
5. Click on `Proceed`.

 <Image src="install/bit-integration-select-plan" alt="How-to-add-calculation" w="1020" h="350" />
 

6. You have to fill up the form for **membership account** to use the premium version of Bit Integrations.

 <Image src="install/membeship-account" alt="plugin install wordpress" w="943" h="500"/>


7. Give your **location** & **payment information**.

 <Image src="install/location-and-payment-bit-integrations" alt="plugin install wordpress" w="574" h="450"/>



8. Pay the amount via any **credit card** or **PayPal**. Also can use **coupon card** if you have any.

 <Image src="install/paypal-bit-integrations" alt="plugin install wordpress" w="565" h="400" />

 <Image src="install/add-coupon-bit-integrations" alt="plugin install wordpress" w="577" h="400"/>


9.  After you will be redirected to **subscription page.**
10. You can check your subscription is activated or not by reloading the page.

 <Image src="install/subcriptions-page-bit-integrations" alt="plugin install wordpress" w="1066" h="380"/>


11. After you will be notified via your email with **download link** , **license key**, **tutorial link** & **documentation link**.

12. Now, go to your subscription dashboard. Here you can see your subscription details. You can also update your plans at any time.

 <Image src="install/upgrade-plan-bit-integrations" alt="plugin install wordpress" w="469" h="438"/>


13. Download the pro version by clicking the option.

 <Image src="install/bit-integrations-pro-version-plugin-download" alt="plugin install wordpress" w="469" h="438"/>

14. Then go to your WordPress dashboard & again navigate to plugin option.
15. Click on `Add New` & select `upload plugin`.

 <Image src="install/upload-plugin-bit-integrations" alt="plugin install wordpress" w="568" h="239"/>
 <Image src="install/plugin-install-bit-integrations" alt="plugin install wordpress" w="803" h="227"/>


16. Click on `Choose File` & select the zip file you have downloaded before. Then click on `Install Now`.

 <Image src="install/click-install-button" alt="plugin install wordpress" w="811" h="150"/>

17. Then press on `Activate Plugin`. After you will see the pro version of Bit Integrations on your plugin list.

 <Image src="install/active-bit-integrations-plugin" alt="plugin install wordpress" w="477" h="207"/>
 <Image src="install/bit-integrations-pro-plugin-active-plugin-list"alt="plugin install wordpress" w="791" h="184"/>


18. Then go to **Bit Integrations** from dashboard & you will find license option. And now copy the key from subscription page & paste the key on license key option.

 <Image src="install/bit-integrations-license-key-dashborad" alt="plugin install wordpress" w="764" h="259" />

19. After clicking `Submit` button you will be shown a successful message.

 <Image src="install/bit-integrations-succefully-activated" alt="plugin install wordpress" w="764" h="259" />

<br />
<br />



**Congratulations! Now you are a pro member of the Bit Integrations. Now you can access pro features, updates & support.**
