**_To integrate with ActiveCampaign follow, the below steps._**

## Process of Authorization and Integration

1. At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **_“Create Integrations”_** button.

   <Image
     src="active-campaign/_create-integrations"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

2. Now select the Ninja Forms that you want to integrate with ActiveCampaign.

   <Image
     src="active-campaign/_select-trigger"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

3. Select the form name from the Drop-Down list that you want to integrate with ActiveCampaign, . And click on the “Next” button. And select **ActiveCampaign.**

   <Image
     src="active-campaign/_ninja-form"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
   <Image
     src="active-campaign/_select-activecampaign"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

4. Integration Name: Set the proper integration name. Default it's taken “ActiveCampaign”.

   <Image
     src="active-campaign/_integration-name"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

5. Access API URL & API key: To get the URL & key : log in to **ActiveCampaign dashboard**=> **Settings**=> **Developer**=> **Get your API URL & Key**.You have to copy the API URL & Key from the Active Campaign **Developer** section & paste the URL & key on the fields of integration settings.

   <Image
     src="active-campaign/_activecampaign-api-dashbord"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
   <Image
     src="active-campaign/_access-uri-and-api-key"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

6. Now click Authorize, & you are ready to integrate with your ActiveCampaign account.

   <Image
     src="active-campaign/_authorize"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

7. After successful authorization. Select the List from the drop-down option. You have to create a list before on your ActiveCampaign account. You can add a Tag [ also multiple tags] for the Contacts that you have created before on your ActiveCampaign account
   <Image
     src="active-campaign/_activecampaign-list"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
   <Image
     src="active-campaign/_activecampaign-tag"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
8. You will find the Map Field option. You have to map all the necessary form fields with the related ActiveCampaign field. Here ‘Email‘ field is the default for ActiveCampaign.
   <Image
     src="active-campaign/_default-field"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
   <Image
     src="active-campaign/_field-mapping"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />

## Actions

9. You can also update the submitted data by enabling the action called Update ActiveCampaign. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address.
   <Image
     src="active-campaign/_update-actions"
     alt="Active-Campaign"
     w="1020"
     h="350"
   />
10. Now save the integration by clicking the "Finish & Save." Button
    <Image
      src="active-campaign/_finish-and-save"
      alt="Active-Campaign"
      w="1020"
      h="350"
    />
11. We have an option in every integration "Timeline," where you can check integration success or error status.

      src="active-campaign/_integrations-dashbord"
      alt="Active-Campaign"
      w="1020"
      h="350"
    />
    <Image
      src="active-campaign/_timeline"
      alt="Active-Campaign"
      w="1020"
      h="350"
    />
