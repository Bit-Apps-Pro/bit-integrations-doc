> **Note: StudioCart is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="StudioCart" w="1020" h="350" />

2. Now select the **StudioCart** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/studiocart/select-trigger" alt="StudioCart" w="1020" h="350" />

3. Select the Task from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/studiocart/select-form-or-task-name" alt="StudioCart" w="1020" h="350"/>

**☞ New Order Created:** This integration will be triggered when a user orders placed. For example, when user orders are placed, you can use this task to send data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) and other third-party platforms.

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
