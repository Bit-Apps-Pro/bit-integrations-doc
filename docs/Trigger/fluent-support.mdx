> **Note: Fluent Support is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **Create Integration** button.

   <Image src="trigger/_create-integrations" alt="Fluent Support" w="1020" h="350" />

2. Now select the **Fluent Support** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/fluent-support/select-trigger" alt="Fluent Support" w="1020" h="350" />

3.	Bit integrations will automatically generate a **webhook URL** & you have to copy that link. And click on the **`fetch`** button. 
   -	Now navigate to Fluent Support Workflows
   -	Create a workflow
   - Type your workflow name, select workflow type automatic, and click on the continue button
   -	Select the workflow trigger (On Ticket Creation/ On Customer Response) That you want to integrate with Bit Integrations Actions
   -	Then select workflow action (Trigger Outgoing Webhook) 
   -	Paste the copied Webhook URL into the webhook URL field
   -	Input the action title and click on the Save button, then publish the workflow
   -	if you Select workflow trigger On Ticket Creation/On Customer Response, then create a new ticket/customer
   -	Go to the Bit integration page and click on the **`set action`** button.

   <Video src="fluent-support-fetch-data" alt="Fluent Support" w="1020" h="350" />
   
> **Note:** Do not copy the webhook URL until the Fetch button is activated.
> **Note: You must install the `Fluent Support pro` plugin to set workflow in Fluent Support.**

<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
