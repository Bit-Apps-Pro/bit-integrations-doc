> **Note: Groundhogg is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="Groundhogg" w="1020" h="350" />

2. Now select the **Groundhogg** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/groundhogg/select-trigger" alt="Groundhogg" w="1020" h="350" />

3. Select the form from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/groundhogg/select-form-or-task-name" alt="Groundhogg" w="1020" h="350"/>
   

**Contact-Create:** This integration will be triggered when creating a new contact.  
*For example,* when an admin creates a contact, you can use this task to send the created contact data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**Add-Tag-To-Contact:** This integration will be triggered when a contact will be added to the tag. 
*For example,* When an admin adds a contact to the tag, you can use this task to send the added contact data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**Remove-Tag-From-Contact:** This integration will be triggered when a contact will be remove to the tag. 
*For example,* When an admin removes a contact to the tag, you can use this task to send the added contact data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
