> **Note: User Registration is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="User Registration" w="1020" h="350" />

2. Now select the **User Registration** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/user-reg/_select-trigger" alt="User Registration" w="1020" h="350" />

3. Select the Task( **`Create New User`** **`User Profile Update`**) from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/user-reg/selec-form-or-task-name" alt="User Registration" w="1020" h="350"/>

**☞ Create New User:** This integration will be triggered when a user register or the admin create a new user.   
*For example,* when an admin creates a new user, and you can use this task to send user data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), CRM, Email marketing tools, and other third-party platforms.

**☞ User Profile Update:** This integration will be triggered when a user edits the profile, or the admin edits the user profile.   
*For example,* when an admin edits the user profile, you can use this task to send edited user data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), CRM, Email marketing tools, and other third-party platforms.

**☞ User Login:** This integration will be triggered when a user login the website.    
*For example,* when a user login the website, you can use this task to send login user data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), CRM, Email marketing tools, and other third-party platforms.

**☞ User reset password:** This integration will be triggered when a user reset the password.   
*For example,* when a user reset the password, you can use this task to send the reset password user data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), CRM, Email marketing tools, and other third-party platforms.

**☞ User delete account:** This integration will be triggered when an admin delete the user.   
*For example,* when an admin delete the user, you can use this task to send the deleted user data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), Email marketing tools, and other third-party platforms.

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
