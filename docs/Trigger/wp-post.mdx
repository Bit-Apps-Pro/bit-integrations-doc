> **Note: WP Post is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **Create Integration** button.

   <Image src="trigger/_create-integrations" alt="WP Post" w="1020" h="350" />

2. Now select the **WP Post** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/wp-post/_select-trigger" alt="WP Post" w="1020" h="350" />

3. Select the Task( `Create a new post`, `Updated a post`, `Delete a post`, `User views a post`, `User comments on a post`, `Change post status`) And Select the post types the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **Next** button.

   <Image src="trigger/wp-post/select-form-or-task-name" alt="WP Post" w="1020" h="350"/>


**☞ Create a new post:** This integration will be triggered when a user creates a new post.   
*For example,* when an user creates a new post, and you can use this task to send post data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.

**☞ Updated a post:** This integration will be triggered when a user updates a post.   
*For example,* when an user update a post, and you can use this task to send updated post data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.

**☞ Delete a post:** This integration will be triggered when a user deletes a post.   
*For example,* when an user delete a post, and you can use this task to send deleted post data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.

**☞ User views a post:** This integration will be triggered when a user views a post.      
*For example,* when an user views a post, and you can use this task to send views post data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.

**☞ User comments on a post:** This integration will be triggered when a user comments on a post.   
*For example,* when an user comments on a post, and you can use this task to send comments data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.

**☞ Change post status:** This integration will be triggered when a user changes post status.   
*For example,*When a user changes a post's status (published to draft), and you can use this task to send post data to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet) CRM, [Email](https://docs.bit-integrations.bitapps.pro/integrations/email), and other third-party platforms.


:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
