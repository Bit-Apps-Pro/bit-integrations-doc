**_To integrate with RapidMail follow, the below steps._**

## Process of Authorization and Integration

1. At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **“Create Integrations”** button.

   <Image src="RapidMail/Create Integration" alt="RapidMail" w="1020" h="350" />

2. Now select WPForms that you want to integrate with RapidMail.

   <Image
     src="RapidMail/Select Contact Form 7"
     alt="RapidMail"
     w="1020"
     h="350"
   />

3. Select the form name from the Drop-Down list that you want to integrate with RapidMail. And click on the “Next” button. And select RapidMail.

   <Image src="RapidMail/WP-Forms" alt="RapidMail" w="1020" h="350" />
   <Image src="RapidMail/Select Rapidmail" alt="RapidMail" w="1020" h="350" />

4. Integration Name: Set the proper integration name Default it's taken “Rapidmail”.

   <Image src="RapidMail/Integration Name" alt="RapidMail" w="1020" h="350" />

5. User Name & Password: To get the User Name & Password log in to **RapidMail dashboard**=> **Profile**=> **API**=> Click on the "Add New API User Button" Get your User Name & Password. You have to copy the User Name & Password from the RapidMail & paste the User Name & Password on the fields of integration setting. And click on the "Authorize" button.

   <Image
     src="RapidMail/Rapidmail API Dashbord"
     alt="RapidMail"
     w="1020"
     h="350"
   />
   <Image src="RapidMail/Authorize" alt="RapidMail" w="1020" h="350" />

6. Select the Recipient from the Drop-Down list. You have to create a Recipient list before on your RapidMail accounts. Map the mandatory field of RapidMail with the corresponding fields. And click on the Next button.

   <Image src="RapidMail/Recipient List" alt="RapidMail" w="1020" h="350" />
   <Image src="RapidMail/Field Mapping" alt="RapidMail" w="1020" h="350" />
   <Image src="RapidMail/Next" alt="RapidMail" w="1020" h="350" />

7. Now save the integration by clicking the "Finish & Save." Button

   <Image src="RapidMail/Finish and Save" alt="RapidMail" w="1020" h="350" />

8. We have an option in every integration "Timeline," where you can check integration success or error status.
   <Image
     src="RapidMail/Integrations Dashbord"
     alt="RapidMail"
     w="1020"
     h="350"
   />
   <Image src="RapidMail/Timeline" alt="RapidMail" w="1020" h="350" />
