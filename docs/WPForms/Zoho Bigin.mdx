***To integrate with Zoho Bigin follow, the below steps.***


## Process of Authorization and Integration
1.	At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **“Create Integrations”** button.
 <Image src="Zoho Bigin/Create Integration" alt="Zoho Bigin" w="1020" h="350" />


2. Now select WPForms that you want to integrate with Zoho Bigin.
 <Image src="Zoho Bigin/Select Contact Form 7" alt="Zoho Bigin" w="1020" h="350" />

3.	Select the form name from the Drop-Down list that you want to integrate with Zoho Bigin. And click on the **“Next” button**. And select **Zoho Bigin.**
 <Image src="Zoho Bigin/WP-Forms" alt="Zoho Bigin" w="1020" h="350" />
 <Image src="Zoho Bigin/select zoho bigin" alt="Zoho Bigin" w="1020" h="350" />

4.	Integration Name: Set the proper integration name Default it's taken “Zoho Bigin”.
 <Image src="Zoho Bigin/integration name" alt="Zoho Bigin" w="1020" h="350" />

5. Select a data center from the dropdown. Then click on the *Zoho API Console*. Navigate to **API Console-**> **Get Started**-> **Server-based Applications.** 
 <Image src="Zoho Bigin/select data center" alt="Zoho Bigin" w="1020" h="350" />
 <Image src="Zoho Bigin/get started" alt="Zoho Bigin" w="1020" h="350" />
 <Image src="Zoho Bigin/server based applications" alt="Zoho Bigin" w="1020" h="350" />


6.	Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click **CREATE** button
 <Image src="Zoho Bigin/create api" alt="Zoho Bigin" w="1020" h="350" />

 
7.	A Client ID and a Client Secret will be generated. Then go to the Client Secret tab
 <Image src="Zoho Bigin/client id and serect" alt="Zoho Bigin" w="1020" h="300" />

8.	Then copy the Client ID and Client Secret and paste it to the Bit Form integration page. Then click on the **Authorize** button (wait for a few seconds). After the successful authorization click on the **Next** button
 <Image src="Zoho Bigin/authorize" alt="Zoho Bigin" w="1020" h="300" />

9.	Select a **Module** from the drop-down list. Map all the mandatory Recruit fields to the corresponding fields in your form.
 <Image src="Zoho Bigin/select module" alt="Zoho Bigin" w="1020" h="300" />
 <Image src="Zoho Bigin/field mapping" alt="Zoho Bigin" w="1020" h="300" />

 ## Actions

**Workflow:** Select this option to trigger the Zoho Bigin workflow for the selected module. If you didn’t select this option Zoho Bigin workflow for the selected module doesn’t work.
 <Image src="Zoho Bigin/workflow actions" alt="Zoho Bigin" w="1020" h="350" />

**Attachment:** To Select this option you can send file upload field data to Zoho Bigin related list Attachments. You can also send file upload field data to Zoho Bigin via normal field mapping). Please select the CRM file upload field from the drop-down which you want to send data.
 <Image src="Zoho Bigin/attachment actions" alt="Zoho Bigin" w="1020" h="350" />


**Approval:** Select this option to automatically send form data to your Zoho Bigin approval list instead of adding them.
 <Image src="Zoho Bigin/approval actions" alt="Zoho Bigin" w="1020" h="350" />

**Add Note:** Select this action to add a note from WPForms to Zoho Bigin account automatically.
 <Image src="Zoho Bigin/add notes actions" alt="Zoho Bigin" w="1020" h="350" />

**Upload Photo:** You can update any contact’s profile photo through our Bit Form to your Zoho Bigin account. For that, hoover on the action called “Upload Photo“. After mapping the relevant fields, click the action ‘Upload Photo’ to upload a photo from Bit Form to Zoho Bigin. Now click on ‘Save’. Whenever the user fills-up the form on Bit Form, the image will be automatically uploaded as a profile photo to the Zoho Bigin account.
 <Image src="Zoho Bigin/upload photo actions" alt="Zoho Bigin" w="1020" h="350" />

## Save Integration
1.	After giving all the required information, click on **Finish & Save**. Now your created **Zoho Bigin API** is ready for implementation.
