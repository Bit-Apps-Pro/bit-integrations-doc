***To integrate with Zoho Campaign follow, the below steps.***


## Process of Authorization and Integration
1. At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **“Create Integrations”** button.
 <Image src="zoho-campaigns/_create-integrations" alt="Zoho Campaign" w="1020" h="350" />


2. Now select WPForms that you want to integrate with Zoho Campaign.
 <Image src="zoho-campaigns/_select-contact-form-7" alt="Zoho Campaign" w="1020" h="350" />

3. Select the form name from the Drop-Down list that you want to integrate with Zoho Campaign. And click on the **“Next” button**. And select **Zoho Campaign.**
 <Image src="zoho-campaigns/_wp-forms" alt="Zoho Campaign" w="1020" h="350" />
 <Image src="zoho-campaigns/_select-zoho-campaigns" alt="Zoho Campaign" w="1020" h="350" />

4. Integration Name: Set the proper integration name Default it's taken “Zoho Campaigns”.
 <Image src="zoho-campaigns/_integration-name" alt="Zoho Campaign" w="1020" h="350" />

5. Select a data center from the dropdown. Then click on the *Zoho API Console*. Navigate to **API Console-**> **Get Started**-> **Server-based Applications.** 
 <Image src="zoho-campaigns/_select-data-center" alt="Zoho Campaign" w="1020" h="350" />
 <Image src="zoho-campaigns/_get-started" alt="Zoho Campaign" w="1020" h="350" />
 <Image src="zoho-campaigns/_server-based-applications" alt="Zoho Campaign" w="1020" h="350" />


6. Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click **CREATE** button
 <Image src="zoho-campaigns/_create-applications" alt="Zoho Campaign" w="1020" h="350" />


7. A Client ID and a Client Secret will be generated. Then go to the Client Secret tab
 <Image src="zoho-campaigns/_client-id-and-serect" alt="Zoho Campaign" w="1020" h="300" />

8. Then copy the Client ID and Client Secret and paste it to the Bit Integrations integration page. Then click on the **Authorize** button (wait for a few seconds). After the successful authorization click on the **Next** button
 <Image src="zoho-campaigns/_authorize" alt="Zoho Campaign" w="1020" h="300" />

9. Select the contact List and the mandatory field from the dropdown.
 <Image src="zoho-campaigns/_select-list" alt="Zoho Campaign" w="1020" h="300" />


10. When you fill-up the contact list field, you will find the Map Field option. You have to map all the necessary form fields with the related Zoho Campaigns field.
 <Image src="zoho-campaigns/_field-mapping" alt="Zoho Campaign" w="1020" h="300" />


## Save Integration
1. After clicking on Finish & Save, the Zoho Campaign integration is ready.
 <Image src="zoho-campaigns/_save-and-finish" alt="Zoho Campaign" w="1020" h="350" />
 
  **Notice: When you send data in Zoho Campaign an email confirmation message will go to the typed email. The data will not insert before the confirmation. If you want to send data without email confirmation, you have to change some settings in the Zoho Campaign. Have to disable SIGNUP FORM.**
 <Image src="zoho-campaigns/_signup-form" alt="Zoho Campaign" w="1020" h="350" />

  We have an option in every integration "Timeline," where you can check integration success or error status.
