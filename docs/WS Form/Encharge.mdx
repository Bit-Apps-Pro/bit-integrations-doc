***To integrate with Encharge follow, the below steps.***


## Process of Authorization and Integration
1.	At First, Open your Bit Integrations Dashboard and create an integrations by clicking the **“Create Integrations”** button.
 <Image src="encharge/_create-integration" alt="Encharge" w="1020" h="350" />


2. Now select WS Form that you want to integrate with Encharge.
 <Image src="encharge/_select-contact-form-7" alt="Encharge" w="1020" h="350" />

3. Select the form name from the Drop-Down list that you want to integrate with Encharge. And click on the **“Next” button.** And select Encharge.
 <Image src="encharge/_ws-form" alt="Encharge" w="1020" h="350" />
 <Image src="encharge/_encgarge" alt="Encharge" w="1020" h="350" />

4. Integration Name: Set the proper integration name. Default it's taken “Encharge”.
 <Image src="encharge/_integration-name" alt="Encharge" w="1020" h="350" />

5.	After that, you have to copy the API key from your Encharge account & paste the key on the field. To get the API key, click on **Encharge CONSOL**=> **click on Account icon** => Your account. Now copy the API key and paste it into the integrations dashboard. And Click on **“Authorize”** button
 <Image src="encharge/_encgarge-api-console" alt="Encharge" w="1020" h="350" />
 <Image src="encharge/_api-console-dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="encharge/_authorize" alt="Encharge" w="1020" h="350" />


6. Click on Next to set up the other integration settings ahead.
 
7. After completing the authorization process, you will find the Map Field option. You have to map all the necessary fields with the related Encharge field. By default “Email” field of Encharge will be available. You can also send custom values by mapping your required field
 <Image src="encharge/_field-mapping" alt="Encharge" w="1020" h="350" />

9.	We have an option in every integration "Timeline," where you can check integration success or error status.
 <Image src="encharge/_integrations-dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="encharge/_timeline" alt="Encharge" w="1020" h="350" />
