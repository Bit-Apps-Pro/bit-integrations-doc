**To integrate with Acumbamail follow, the below steps.**

## Authorization Process
> **Note: Acumbamail is only available on the pro version.**

1. Search & select **Acumbamail** from the actions pages.
  <Image src="actions/acumbamail/select-acumbamail" alt="Autonami" w="1020" h="400" />

2. **Integration Name:** Set the integration name “Acumbamail”.
 <Image src="actions/acumbamail/integration-name" alt="Acumbamail" w="1020" h="300" />

3. **Auth Token:** To get the **Auth Token** log in to Acumbamail.  Then Open [**Acumbamail API Doc**](https://acumbamail.com/en/apidoc/) in Customer identifier section you will find the Auth Token. You have to copy Auth Token from the Acumbamail API Doc & paste the Auth Token field of integration settings.
 <Image src="actions/acumbamail/auth-token" alt="Acumbamail" w="1020" h="300" />
 <Image src="actions/acumbamail/get-auth-token-from-acumbamail" alt="Acumbamail" w="1020" h="300" />


4. Now click Authorize, & you are ready to integrate with your **Acumbamail** account.

 <Image src="actions/acumbamail/authorize" alt="Acumbamail" w="1020" h="270" />


5. After successful authorization. Select the Action(`Add/Update Subscriber`, `Delete Subscriber`) from the drop-down option. Also select the List from the drop-down option. You have to create a list before on your Acumbamail account. 

  **A. Add/Update Subscriber:** If you select this option from the drop-down, a new customer will be added to your Acumbamail list, or the user already exists in your list; It will then update the customer based on the email address.

  **B. Delete Subscriber:** If you selected this option from the drop-down, the records would be deleted from your Acumbamail account based on the email address.

 <Image src="actions/acumbamail/select-action" alt="Acumbamail" w="1020" h="300" />
 <Image src="actions/acumbamail/select-list" alt="Acumbamail" w="1020" h="300" />

6. You will find the Map Field option. You have to map all the necessary form/task fields with the related Acumbamail field. Here ***'Email'*** field is the default for Acumbamail. After mapping the desired fields, click on Next button.

 <Image src="actions/acumbamail/default-field" alt="Acumbamail" w="1020" h="300" />
 <Image src="actions/acumbamail/field-mapping" alt="Acumbamail" w="1020" h="300" />

7. After clicking ***Finish & Save*** you can see the integration you have created now. 

 <Image src="actions/acumbamail/finish-and-save" alt="Acumbamail" w="1020" h="250" />a


8.	We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/acumbamail/timeline" alt="Autonami" w="1020" h="350" />
 <Image src="actions/acumbamail/success-or-error-status" alt="Autonami" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/acumbamail/edit-clone-delete" alt="Elastic Email" w="1020" h="350" />
