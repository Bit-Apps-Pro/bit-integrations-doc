***To integrate with Autonami follow, the below steps.***

## Process of Authorization and Integration

1. Search & Autonami Sheets from the actions pages.
  <Image src="actions/autonami/_select-autonami" alt="Autonami" w="1020" h="400" />

2. Before setting up the integration, you must confirm that the Autonami Pro plugin is activated on your site. Otherwise, the integration will not work.

2. Integration Name: Set the proper integration name. Default it's taken **“Autonami”**.
  <Image src="actions/autonami/_integration-name" alt="Autonami" w="1020" h="400" />

3. Now click on the Authorize button to complete authorization settings. Then Click on next button.
  <Image src="actions/autonami/Authorization" alt="Autonami" w="1020" h="400" />


4.	After completing the authorization process, select the contacts list that you have created before on Autonami. You can choose multiple lists from here. You can add a Tag [ also multiple tags] for the contacts that you have created before on your Autonami account. Your Autonami plugin must already have tags.

  <Image src="actions/autonami/List Select" alt="Autonami" w="1020" h="400" />
  <Image src="actions/autonami/Tag Added" alt="Autonami" w="1020" h="400" />

5.	Then you have to map the form fields with Autonami fields. By default, the field is Email. Also, you set any field required on Autonami, and it will be automatically displayed on your map settings. You can set multiple fields from map field settings.

  <Image src="actions/autonami/Map Fields" alt="Autonami" w="1020" h="400" />

## Actions

1.	If the contact already exists in Autonami. It will be skip contact; otherwise, it will update the contact based on email address.

  <Image src="actions/autonami/Action" alt="Autonami" w="1020" h="400" />

2. After clicking **Finish & Save** you can see the integration you have created now.

  <Image src="actions/autonami/Finish & Save" alt="Autonami" w="1020" h="400" />
  <Image src="actions/autonami/Completed List" alt="Autonami" w="1020" h="400" />
