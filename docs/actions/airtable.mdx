> **Note: ConvertKit is available in the free and pro versions.**

***To integrate with convertkit follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **ConvertKit** from the actions pages
 <Image src="actions/convertkit/search_converkit" alt="convertkit" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'ConvertKit'.
 <Image src="actions/convertkit/integration-name" alt="convertkit" w="1020" h="350" />

3.	**Access API Secret Key:** 

 **To Get Access API Secret Key,**  
 -	Login to your ConvertKit account
 -	Now go to **Settings**, Then click **Advanced**
 <Image src="actions/convertkit/api-key" alt="convertkit" w="1020" h="350" />
 <Image src="actions/convertkit/api-secret" alt="convertkit" w="1020" h="350" />

4. You have to copy the API Secret from the ConvertKit account & paste into the Access API Secret Key on the fields of integration settings. And click on aithorize button 

 <Image src="actions/convertkit/authorized" alt="convertkit" w="1020" h="350" />


5. After successful authorization. Select the `Form` from the drop-down option. You have to create a `Form` before on your ConvertKit account. You can add a Tag [ also multiple tags] for the Contacts that you have created before on your ConvertKit account

 <Image src="actions/convertkit/select-form" alt="convertkit" w="1020" h="350" />
 <Image src="actions/convertkit/select-tag" alt="convertkit" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related ConvertKit field. And click on `Next` button.
 <Image src="actions/convertkit/field-mapping" alt="convertkit" w="1020" h="350" />


### Actions

1. You can also update the submitted data by enabling the action called `Update ConvertKit`. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address. 
 <Image src="actions/convertkit/update" alt="convertkit" w="1020" h="350" />

2. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/convertkit/finish-save" alt="convertkit" w="1020" h="350" />

3. Bit Integrations has an option in every integration called **`Timeline`** where you can check integration success or error status.
 <Image src="actions/convertkit/timiline" alt="convertkit" w="1020" h="350" />
 <Image src="actions/convertkit/success-error-status" alt="convertkit" w="1020" h="350" />

4. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/convertkit/edit-clone-delete" alt="convertkit" w="1020" h="350" />
