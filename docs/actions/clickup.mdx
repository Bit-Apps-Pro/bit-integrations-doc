> **Note: ClickUp is available in the free and pro versions.**

ClickUp is a versatile productivity and project management tool designed to facilitate team collaboration and organization. With the help of Bit Integrations, you can easily create a task in your ClickUp account. This integration is compatible with several popular platforms such as WordPress Forms, WooCommerce, LMS, and Membership.

Once a trigger event occurs on any of these platforms, Bit Integrations will use field mapping to automatically generate a task in ClickUp that contains all the relevant data and information from the trigger event.  Say goodbye to the hassle of manual task creation and hello to streamlined productivity!

***To integrate with ClickUp follow, the below steps.***


## Process of Authorization and Integration

1. Search & Select ClickUp from the actions pages.
 <Image src="actions/clickup/select-clickup" alt="ClickUp" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken **ClickUp**.
 <Image src="actions/clickup/integration-name" alt="ClickUp" w="1020" h="350" />

3. **API Key:** To get the ClickUp API Key.
-   Navigate to your personal `Settings`. 
-   Click `Apps` in the left sidebar.
-   Click `Generate` to create your API token.
-   Click `Copy` to copy the key to your clipboard.

 <Video src="clickup-api-key" alt="ClcikUp" w="1020" h="350" />


4. **Select Action:** Select the Action (Create Task) from the drop-down option.
 <Image src="actions/clickup/select-action" alt="ClickUp" w="1020" h="350" />

5. **Select Team:** Select the Team that you have already created on your ClickUp account from the drop-down menu.
 <Image src="actions/clickup/select-team" alt="ClickUp" w="1020" h="350" />

6. **Select Space:** Select the Space that you have already created on your ClickUp account from the drop-down menu.
 <Image src="actions/clickup/select-space" alt="ClickUp" w="1020" h="350" />

7. **Select Folder:** Select the Folder that you have already created on your ClickUp account from the drop-down menu.
 <Image src="actions/clickup/select-folder" alt="ClickUp" w="1020" h="350" />

8. **Select List:** Select the List that you have already created on your ClickUp account from the drop-down menu.
 <Image src="actions/clickup/select-list" alt="ClickUp" w="1020" h="350" />


9. After that, you will find the Map Field option. You have to map all the necessary fields with the related ClickUp field. After mapping the desired fields, click on the `Next` button.
 <Image src="actions/clickup/field-mapping" alt="ClickUp" w="1020" h="350" />

10. Now save the integration by clicking the Finish & Save Button
 <Image src="actions/clickup/finish-save" alt="ClickUp" w="1020" h="350" />

11. Bit Integrations is an option in every integration called `Timeline` where you can check integration success or error status.
 <Image src="actions/clickup/timeline" alt="ClickUp" w="1020" h="350" />
 <Image src="actions/clickup/integration-log" alt="ClickUp" w="1020" h="350" />

12. Also, you can `Edit`, `Clone`, and `Delete` the integration.
 <Image src="actions/clickup/edit-clone-delete" alt="ClickUp" w="1020" h="350" />
