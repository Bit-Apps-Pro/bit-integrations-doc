> **Note: DirectId is available in the free and pro versions.**

DirectId integration via Bit Integrations is a seamless way to send data from your WordPress forms, Page/Theme Builder, WooCommerce, LMS, Membership, and other plugins to your DirectId account. This integration makes email marketing easier and more efficient by automating the process of data transfer, eliminating the need for manual data entry. With Bit Integrations, you can set up the DirectId integration quickly and easily, and start sending data to your DirectId account in just a few clicks. For more details, refer to the integration guide provided.

## Process of Authorization and Integration

1.	Search & Select **DirectId** from the actions pages
 <Image src="actions/direct_iq/select-direct-iq" alt="DirectIq" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken DirectIq.
 <Image src="actions/direct_iq/integrations-name" alt="DirectIq" w="1020" h="350" />

3.	**Client ID & Client Secret:** To get Client Id & Client secret go to your DirectIq account.
-   Open your DirectIq account
-   Now go to Integration from the left side menu
-   Click API

 <Image src="actions/direct_iq/client-id-screct" alt="DirectIq" w="1020" h="350" />

4. After that, you have to copy the Client ID & Client Secret from your DirectIq account & paste them into the integration setting. And Click on `Authorize` button. After successfully authorized done. Click on `Next` to set up the other integration settings ahead.
 <Image src="actions/direct_iq/authorized" alt="DirectIq" w="1020" h="350" />

5.	After completing the authorization process, Select List from the drop-down options. If you don't see any list. Click on the refresh button.
 <Image src="actions/direct_iq/select-list" alt="DirectIq" w="1020" h="350" />

6.	You will find the Map Field option. Map all the necessary fields with the related DirectIq field. The Email field of DirectIq will be available by default, but you can also send custom values by mapping your required field. Click on the refresh button if you don't see any fields.
 <Image src="actions/direct_iq/field-mapping" alt="DirectIq" w="1020" h="350" />

7. Save the integration by clicking the `Finish & Save` button.

 <Image src="actions/direct_iq/finish-save" alt="DirectIq" w="1020" h="350" />

8. Bit Integrations is an option in every integration called **`Timeline`** where you can check integration success or error status.
 <Image src="actions/direct_iq/timeline" alt="DirectIq" w="1020" h="350" />
 <Image src="actions/direct_iq/success-error-status" alt="DirectIq" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/direct_iq/edit-clone-delete" alt="DirectIq" w="1020" h="350" />
