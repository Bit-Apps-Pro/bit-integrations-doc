> **Note: EmailOctopus is available in the free and pro versions.**

***To integrate with EmailOctopus follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **EmailOctopus** from the actions pages
 <Image src="actions/emailoctopus/search-and-select" alt="EmailOctopus" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'EmailOctopus'.
 <Image src="actions/emailoctopus/default-integration-name" alt="EmailOctopus" w="1020" h="350" />

3.	After that, you have to copy the API key from your EmailOctopus account & paste the API key on the field.   
**To get EmailOctopus Api key**
-	Click on [EmailOctopus API keys](https://emailoctopus.com/api-documentation)
-	Click on create key button to create an api key
-	Input your api key name of your preference and click on create button
-	Then copy the api key and paste into api key field of your authorization form. And Click on **Authorize** button.

 <Image src="actions/emailoctopus/click-here" alt="EmailOctopus" w="1020" h="350" />
 <Image src="actions/emailoctopus/visit-api-console" alt="EmailOctopus" w="1020" h="350" />
 <Image src="actions/emailoctopus/authorize" alt="EmailOctopus" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/emailoctopus/authorize-after" alt="EmailOctopus" w="1020" h="350" />

7.	After completing the authorization process, Select the List of your choices from the drop-down options, and you can select multiple lists. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/emailoctopus/lists-select" alt="EmailOctopus" w="1020" h="350" />

8.	You will find the Map Field option. You have to map all the necessary fields with the related EmailOctopus field. By default, **`Email`** field of EmailOctopus will be available. You can also send custom values by mapping your required field and custom field. If you don't see any fields and missing. Click on the `refresh` button.
 
 <Image src="actions/emailoctopus/emailoctopus-field-map" alt="EmailOctopus" w="1020" h="350" />

6. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/emailoctopus/finish-save" alt="EmailOctopus" w="1020" h="350" />


## Actions 
- **Add Tags:** You can enable the action Add Tags Field to add the tags of the contact. In addition, you can add multiple tags from the Drop-Down list.
 <Image src="actions/emailoctopus/add-tags" alt="EmailOctopus" w="1020" h="350" />
  
- **Update Contact:** If the user already exists on your EmailOctopus account, It will update the contacts with other field information based on the email address.
 <Image src="actions/emailoctopus/update-contact" alt="EmailOctopus" w="1020" h="350" />
 
- **Unsubscribe contact:** The initial status of the contact: SUBSCRIBED. If you enable this option, the contact will receive an invitation letter to subscribe. When the user accepts the invite, the user will be added to the subscriber list.
 <Image src="actions/emailoctopus/unsubscribe-contact" alt="EmailOctopus" w="1020" h="350" />


7. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/emailoctopus/emailoctopus-timeline" alt="EmailOctopus" w="1020" h="350" />
 <Image src="actions/emailoctopus/emailoctopus-log" alt="EmailOctopus" w="1020" h="350" />

8. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/emailoctopus/emailoctopus-option" alt="EmailOctopus" w="1020" h="350" />
