> **Note: Encharge is only available on the pro version.**


***To integrate with Encharge follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Encharge** from the actions pages
 <Image src="actions/encharge/_encgarge" alt="Encharge" w="1020" h="350" />

4. Integration Name: Set the proper integration name. Default it's taken “Encharge”.
 <Image src="actions/encharge/_integration-name" alt="Encharge" w="1020" h="350" />

5.	After that, you have to copy the API key from your Encharge account & paste the key on the field. To get the API key, click on **Encharge CONSOL**=> **click on Account icon** => Your account. Now copy the API key and paste it into the integrations dashboard. And Click on **“Authorize”** button
 <Image src="actions/encharge/_encgarge-api-console" alt="Encharge" w="1020" h="350" />
 <Image src="actions/encharge/_api-console-dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="actions/encharge/_authorize" alt="Encharge" w="1020" h="350" />


6. Click on Next to set up the other integration settings ahead.
 
7. After completing the authorization process, you will find the Map Field option. You have to map all the necessary fields with the related Encharge field. By default “Email” field of Encharge will be available. You can also send custom values by mapping your required field
 <Image src="actions/encharge/_field-mapping" alt="Encharge" w="1020" h="350" />

9.	We have an option in every integration "Timeline," where you can check integration success or error status.
 <Image src="actions/encharge/_integrations-dashbord" alt="Encharge" w="1020" h="350" />
 <Image src="actions/encharge/_timeline" alt="Encharge" w="1020" h="350" />
