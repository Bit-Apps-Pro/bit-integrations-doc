> **Note: Freshdesk is available in the free and pro versions.**

***To integrate with Freshdesk follow, the below steps.***


## Process of Authorization

1.	Search & Select **Freshdesk** from the actions pages
 <Image src="actions/freshdesk/_select-freshdesk" alt="Freshdesk" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Freshdesk'.
 <Image src="actions/freshdesk/_integration-name" alt="Freshdesk" w="1020" h="350" />

3.	**Your App Domain and App API key:** To get App Domain and API go to your Freshdesk account. Copy the App Domain from the **URL** and paste it into Your **App Domain Field**. To get the **API key** to navigate to your Freshdesk **Profile settings**. There you will find the API key on top-right. Copy the **API key** And paste it into App API key field in integration settings. 

 <Image src="actions/freshdesk/_app-domain-url" alt="Freshdesk" w="1020" h="350" />
 <Image src="actions/freshdesk/_api-key" alt="Freshdesk" w="1020" h="350" />
 <Image src="actions/freshdesk/_copy-and-paste-app-domain-and-api" alt="Freshdesk" w="1020" h="350" />


4.	Now click on **`Authorize`** Button after successfully authorizing click on the **`Next`** button. Then you will see the map fields option and map all fields mandatory with the corresponding fields in your form/task.

 <Image src="actions/freshdesk/_authorize" alt="Freshdesk" w="1020" h="350" />

5.	Map all the mandatory Freshdesk fields to the corresponding fields in your form/task.
 <Image src="actions/freshdesk/_field-mapping" alt="Freshdesk" w="1020" h="350" />

6. **Show contact field:** Also you can map Freshdesk contact with your form/task field by enabling this option. And you have to map all the mandatory with corresponding fields.
 <Image src="actions/freshdesk/_map-contact-field" alt="Freshdesk" w="1020" h="350" />

## Actions
1. **Status:** Status is mandatory in Freshdesk. You have to select the status from actions
 <Image src="actions/freshdesk/_status-action" alt="Freshdesk" w="1020" h="350" />

2. **Priority:** Also Priority is mandatory in Freshdesk. You have to select the status from actions.
 <Image src="actions/freshdesk/_priority-actions" alt="Freshdesk" w="1020" h="350" />

3. **Photo:** You can add any contact’s profile photo Ticket profile photo through our Bit Integration to your Freshdesk. For that, hoover on the action called “Photo“. After mapping the relevant field, click the action **‘Photp’** to upload a photo from Form/Task to your Freshdesk. The image will be automatically uploaded as a profile photo to the Freshdesk contact’s or Ticket.
 <Image src="actions/freshdesk/_file-actions" alt="Freshdesk" w="1020" h="350" />

4. **File:** This action sends attachments from your Form/Task to Freshdesk. After mapping the relevant fields, click the action **‘File‘** to send any file from your Form/Task  to Freshdesk. Now click on **‘Save’**. You can now directly check the file attachments from your Freshdesk.
 <Image src="actions/freshdesk/_file-actions" alt="Freshdesk" w="1020" h="350" />

5. **Update:** You can also update the submitted data by enabling the action called Update. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled then the new data will be updated regarding that email address.
 <Image src="actions/freshdesk/_update-actions" alt="Freshdesk" w="1020" h="350" />


9. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/freshdesk/_finish-and-save" alt="Freshdesk" w="1020" h="350" />

10. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/freshdesk/_timeline" alt="Freshdesk" w="1020" h="350" />
 <Image src="actions/freshdesk/_sucess-and-error-massage" alt="Freshdesk" w="1020" h="350" />
