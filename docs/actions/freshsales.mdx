> **Note: FreshSales is available in the free and pro versions.**

FreshSales integrations via Bit Integrations is a seamless way to connect your FreshSales account with other business tools and applications. With a wide range of integrations available. You can send data from your WordPress forms, Page/Theme Builder, WooCommerce, LMS, Membership, and other plugins to your FreshSales account. For more details, refer to the integration guide provided.

## Process of Authorization and Integration

1.	Search & Select **FreshSales** from the actions pages
 <Image src="actions/fresh-sales/select-freshsales" alt="FreshSales" w="1020" h="350" />

2. **Bundle Alias & API Token:** To get Bundle Alias & API Token.
-   Open your FreshSales account.
-   Click on the Profile icon then Navigate to Setting
-   Navigate to API Settings

 <Image src="actions/fresh-sales/alias-and-token" alt="FreshSales" w="1020" h="350" />
 <Image src="actions/fresh-sales/freshsales-setting" alt="FreshSales" w="1020" h="350" />
 <Image src="actions/fresh-sales/freshsales-api-ket-alias" alt="FreshSales" w="1020" h="350" />

3. After that, you have to copy the Bundle Alias & API Token from your FreshSales account & paste them into the integration setting. And Click on `Authorize` button. After successfully authorized done. Click on `Next` to set up the other integration settings ahead.
 <Image src="actions/fresh-sales/authorized" alt="FreshSales" w="1020" h="350" />

4.	**Module:** Choose the desired module by selecting from the available options in the drop-down list.
 <Image src="actions/fresh-sales/select-module" alt="FreshSales" w="1020" h="350" />

5.	**Account View:** Select the relevant module view from the drop-down options. Make sure to choose the appropriate view for the selected module.
 <Image src="actions/fresh-sales/select-account-view" alt="FreshSales" w="1020" h="350" />

6.	**Related Module:** This option allows you to fetch records from a submodule (related list) that you can associate with a record in a parent module.
 <Image src="actions/fresh-sales/select-account" alt="FreshSales" w="1020" h="350" />



7.	You will find the Map Field option. You have to map all the necessary fields with the related FreshSales field. After mapping the desired fields, click on the `Next` button.
 <Image src="actions/fresh-sales/field-mapping" alt="FreshSales" w="1020" h="350" />
<details>
      <summary>
       <b>âœ… Available FreshSales mapping fields</b>
      </summary>
      <div>
         <ul>
            <li>Respective Module Default fields</li>
            <li>Module's Custom fields</li>
         </ul>
      </div>
</details>

8. Now save the integration by clicking the Finish & Save Button

 <Image src="actions/fresh-sales/finish-save" alt="FreshSales" w="1020" h="350" />

9. Bit Integrations is an option in every integration called **`Timeline`** where you can check integration success or error status.
 <Image src="actions/fresh-sales/timeline" alt="FreshSales" w="1020" h="350" />
 <Image src="actions/fresh-sales/integration-log" alt="FreshSales" w="1020" h="350" />

10. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/fresh-sales/edit-clone-delete" alt="FreshSales" w="1020" h="350" />
