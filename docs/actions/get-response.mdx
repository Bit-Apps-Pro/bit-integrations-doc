> **Note: Get Response is available in the free and pro versions.**

***To integrate with Get Response follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Get Response** from the actions pages
 <Image src="actions/get-response/search-and-select" alt="Get Response" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Get Response'.
 <Image src="actions/get-response/default-integration-name" alt="Get Response" w="1020" h="350" />

3.	After that, you have to copy the API key from your GetResponse account & paste the API key on the field. To get the API key, click on the GetResponse API Token or    
**To get Get Response API key**
-	Goto tools and click on Integration and API.
-	Click on API tab and click on generate API key
- Copy the Token and paste into API Key field of your authorization form. And Click on **Authorize** button
 <Image src="actions/get-response/click-here" alt="Get Response" w="1020" h="350" />
 <Image src="actions/get-response/visit-api-console" alt="Get Response" w="1020" h="350" />
 <Image src="actions/get-response/authorize" alt="Get Response" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/get-response/authorize-after" alt="Get Response" w="1020" h="350" />

5.	After completing the authorization process, Select the `List` from the drop-down option from your choices dropdown option. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/get-response/select-list" alt="Get Response" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related GetResponse field. By default, **`Email`** field of GetResponse will be available. You can also send custom values by mapping your required field. Then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/get-response/get-response-field-map" alt="Get Response" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/get-response/finish-save" alt="Get Response" w="1020" h="350" />

## Actions
1.	**Tag:** You can enable the action Add Tags Field to add the tags of the contact. In addition, you can add multiple tags from the Drop-Down list. If you don't see any tags. Click on the `refresh` button.
 <Image src="actions/get-response/tag" alt="Get Response" w="1020" h="350" />

2.	**Update Contact:** You can also update the submitted data by enabling the action called Update contact. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address.
 <Image src="actions/get-response/update=contact" alt="Get Response" w="1020" h="350" />

3. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/get-response/get-response-timeline" alt="Get Response" w="1020" h="350" />
 <Image src="actions/get-response/get-response-log" alt="Get Response" w="1020" h="350" />

4. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/get-response/get-response-option" alt="Get Response" w="1020" h="350" />
