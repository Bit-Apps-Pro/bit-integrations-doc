***To integrate with Getgist follow, the below steps.***


## Process of Authorization and Integration
> **Note: Getgist is only available on the pro version.**

1.	Search & Select **Getgist** from the actions pages
 <Image src="actions/getgist/search-and-select" alt="GetGist" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'GetGist'.
 <Image src="actions/getgist/default-integration-name" alt="GetGist" w="1020" h="350" />

3.	After that, you have to copy the API key from your Getgist account & paste the key on the field. To get the API key, click on **Getgist API Console** or **go to your Getgist dashboard =>click on Account Setting => Setting =>API Key**. Now copy the API key and paste it into the integrations dashboard. And Click on **`“Authorize”`** button.

 <Image src="actions/getgist/visit-api-console" alt="GetGist" w="1020" h="350" />
 <Image src="actions/getgist/getgist-account-api-copy" alt="GetGist" w="1020" h="350" />
 <Image src="actions/getgist/authorize" alt="GetGist" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.

5. After completing the authorization process, Select the User Type from the drop-down option `User` or `Lead`.

6.	You will find the Map Field option. You have to map all the necessary fields with the related Getgist field. By default, **`“Email”`** field of Getgist will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/getgist/getgist-field-map" alt="GetGist" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/getgist/finish-save" alt="GetGist" w="1020" h="350" />

8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/getgist/getgist-timeline" alt="GetGist" w="1020" h="350" />
 <Image src="actions/getgist/getgist-log" alt="GetGist" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/getgist/getgist-option" alt="GetGist" w="1020" h="350" />
