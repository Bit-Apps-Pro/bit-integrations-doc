> **Note: GiveWp is available in the free and pro versions.**

***To integrate with GiveWp follow, the below steps.***

## Process of Authorization and Integration


1.	Search & Select **GiveWp** from the actions pages
 <Image src="actions/givewp/select-action" alt="GiveWp" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken **GiveWp**.

3. Now, click on the `connect` button to complete authorization settings.

 <Image src="actions/givewp/connect-button" alt="GiveWp" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.

 <Image src="actions/givewp/authorize" alt="GiveWp" w="1020" h="350" />

## Actions List
### Create a donor
If you select this option from the drop-down. When users submit/complete the form/task, create a donor to GiveWP.

1.	After the authorization process, select the Action as **create a donor**.
  
 <Image src="actions/givewp/select-dropdown-add-donor" alt="GiveWp" w="1020" h="350" />

2.	You will find the Map Field option. You have to map all the necessary fields with the related GiveWp field. By default, **Email** field of GiveWP will be available. You can also send custom values by mapping your required field. Now, click on the `Next` button to move forward for further settings.
 <Image src="actions/givewp/givewp-filed-map" alt="GiveWp" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/givewp/finish-save" alt="GiveWp" w="1020" h="350" />

 <Image src="actions/givewp/givewp-timeline-log" alt="GiveWp" w="1020" h="350" />
