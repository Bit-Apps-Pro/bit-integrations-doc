> **Note: Groundhogg is only available on the pro version.**

***To integrate with Groundhogg follow, the below steps.***


## Process of Authorization and Integration


1.	Search & Select **Groundhogg** from the actions pages
 <Image src="actions/groundhogg/select-action" alt="Groundhogg" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken Groundhogg.
 <Image src="actions/groundhogg/integration-name" alt="Groundhogg" w="1020" h="350" />

3.	After that, you have to copy the Groundhogg Public key and Token from your Groundhogg account & paste the key on the field.   
**To get the Groundhogg Member URL and Password,**

- Go to groundhogg dashboard 
- Click on Setting option and go to `API`
- Select a user, you Want to create an API key for the user. And click on `Generate New API` Keys button.
- Now copy the `Public key` and `Token`, paste it into the integrations dashboard. And Click on **`Authorize`** button

 <Image src="actions/groundhogg/api-setting" alt="Groundhogg" w="1020" h="350" />
 <Image src="actions/groundhogg/search-api" alt="Groundhogg" w="1020" h="350" />
 <Image src="actions/groundhogg/authorize" alt="Groundhogg" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.

 <Image src="actions/groundhogg/after-authorize" alt="Groundhogg" w="1020" h="350" />

### Integration Settings for Create Contact
1.	After the authorization process, select the `Action` as **Create Contact**. 

 <Image src="actions/groundhogg/select-dropdown-create-contact" alt="Groundhogg" w="1020" h="350" />

2.	When you select the Action, you will see the `Map Fields` option where one fields **Email**, are mapped by default. You can map multiple fields according to your choice for the Action. And To add more groundhog meta fields enable the **`Add meta fields`** option. Then click on the `Next` button to move forward for further settings.

 <Image src="actions/groundhogg/map-fields" alt="Groundhogg" w="1020" h="350" />
 <Image src="actions/groundhogg/map-fields-next" alt="Groundhogg" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/groundhogg/finish-save" alt="Groundhogg" w="1020" h="350" />

### Integration Settings for Add Tag to user
1.	After the authorization process, select the Action from the drop-down option **Add Tag to user**. 
 <Image src="actions/groundhogg/select-dropdown-add-tag-to-user" alt="Groundhogg" w="1020" h="350" />

2.	When you select the Action, you will see the `Emails` and `All tags` option. Click on `Emails` option and select the Email fields from the drop-down option, you can add Form Email fields.
 <Image src="actions/groundhogg/email-fields-map" alt="Groundhogg" w="1020" h="350" />

3.	Click on `All Tags` option and select the tags from the drop-down, you can add multiple tag. Then click on the `Next` button to move forward for further settings. If you don't see any tags. Click on the `refresh` button.

 <Image src="actions/groundhogg/tag-fields-map" alt="Groundhogg" w="1020" h="350" />

4. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/groundhogg/finish-save" alt="Groundhogg" w="1020" h="350" />

## Actions
**Add Tags:** You can enable the action Add Tags Field to add the tags of the contact. In addition, you can add multiple tags from the Drop-Down list.
 <Image src="actions/groundhogg/add-tags" alt="Groundhogg" w="1020" h="350" />

**Option status:** In groundhogg the option status default is unconfirmed. You can change the action opt-in status Field to change the status of the contact opt-in.

 <Image src="actions/groundhogg/optin-status" alt="Groundhogg" w="1020" h="350" />
