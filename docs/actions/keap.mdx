> **Note: Keep is only available on the pro version.**

***To integrate with Keap follow, the below steps.***


## Process of Authorization and Integration

1. Search & Select **Keap** from the actions pages
 <Image src="actions/keap/_select-keap" alt="Keep" w="1020" h="350" />
 
2. Integration Name: Set the proper integration name. Default it's taken **“Keap”**.
 <Image src="actions/keap/_integration-name" alt="Keep" w="1020" h="350" />

3.	**Client ID & Client Secret:** To get a new **Client Id & Client secret** go to **Keap API CONSOLE** > **Create An App**. In the App, the field gives the proper app name. And APIs option enables the APIs by clicking **"Enable".** and click on **"Save"** button.
 <Image src="actions/keap/_new-app" alt="Keep" w="1020" h="350" />
 <Image src="actions/keap/_enable-apis" alt="Keep" w="1020" h="350" />



4. After completing these steps a Client ID and a Client Secret will be generated. 
 <Image src="actions/keap/_copy-the-apis" alt="Keep" w="1020" h="350" />

5. Then You have to copy the **Client ID & Client secret** from Keap & paste the **Client ID & Client secret** on the fields of integration settings. Then click on the **`Authorize`** button (wait for a few seconds). After the successful authorization click on the **`Next`** button
 <Image src="actions/keap/_authorize" alt="Keep" w="1020" h="350" />
 <Image src="actions/keap/_allow" alt="Keep" w="1020" h="350" />


6. Now you have to map all the mandatory Keap fields to the corresponding fields in your form/task. Then click on next button.
 <Image src="actions/keap/_field-mapping" alt="Keep" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save.`** Button
    <Image src="actions/keap/_finish-and-save" alt="keap" w="1020" h="300" />


8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
   <Image src="actions/keap/_timeline" alt="Keep" w="1020" h="350" />
   <Image src="actions/keap/_sucess-or-error-massage" alt="Keep" w="1020" h="350" />
