> **Note: Kirim Email is available in the free and pro versions.**

**To integrate with Kirim Email follow, the below steps.**

## Process of Authorization and Integration

1. Search & Select **Kirim Email** from the actions pages
   <Image src="actions/kirim_email/search_kirim"alt="kirim_email"w="1020"h="350"/>

2. **Integration Name:** Set the proper integration name. Default it's taken “Kirim Email”.
   <Image src="actions/kirim_email/integration_name"alt="kirim_email"w="1020"h="350"/>

3. **Your username & App API key:** To get the username & API key, log in to **Kirim Email dashboard=> Profile=> Account Setting=> API.** To get your username & API key. You have to copy the username & API key from the Kirim Email API section & paste the username & API key into the fields of integration settings.
   <Image src="actions/kirim_email/integration_name"alt="kirim_email"w="1020"h="350"/>
   <Image src="actions/kirim_email/account_setting"alt="kirim_email"w="1020"h="350"/>
   <Image src="actions/kirim_email/username_and_api_token"alt="kirim_email"w="1020"h="350"/>
   <Image src="actions/kirim_email/api_token"alt="kirim_email"w="1020"h="350"/>



4. Now click Authorize, & you are ready to integrate with your **Kirim Email** account

   <Image src="actions/kirim_email/authorize"alt="kirim_email"w="1020"h="350"/>


5. After successful authorization, select the Action **`Add Subscriber`, `Remove Subscriber`** from the drop-down option. Also, choose the list of the drop-down option. You have to create a list before on your Kirim Email account.

  **A. Add Subscriber:** If you select this option from the drop-down, a new Subscriber will be added to your Kirim Account's list, or the user already exists in your list;  It will then update the customer with other field information based on the email address.
  
  **B. Remove Subscriber:** If you selected this option from the drop-down, the records would be deleted from your Kirim Account account based on the email address. 
   <Image src="actions/kirim_email/select_action"alt="kirim_email"w="1020"h="350"/>
   <Image src="actions/kirim_email/select_list"alt="kirim_email"w="1020"h="350"/>


6. You will find the Map Field option. You have to map all the necessary fields with the related Kirim field. Here **‘Email‘ and ‘Name‘** field is the default for Kirim Account. After mapping the desired fields, click on Next button.
   <Image src="actions/kirim_email/field_mapping"alt="kirim_email"w="1020"h="350"/>

7. After clicking **Finish & Save** you can see the integration you have created now.
   <Image src="actions/kirim_email/finish_save"alt="kirim_email"w="1020"h="350"/>

8.	Bit Integrations has an option in every integration called **`Timeline`** where you can check integration success or error status.
 <Image src="actions/kirim_email/timeline" alt="kirim_email" w="1020" h="350" />
 <Image src="actions/kirim_email/success_error" alt="kirim_email" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/kirim_email/edit_clone_delete" alt="kirim_email" w="1020" h="350" />
