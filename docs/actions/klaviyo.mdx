> **Note: Klaviyo is available in the free and pro versions.**

***To integrate with Klaviyo follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Klaviyo** from the actions pages
 <Image src="actions/klaviyo/select-action" alt="Klaviyo" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken Klaviyo.
 <Image src="actions/klaviyo/integration-name" alt="Klaviyo" w="1020" h="350" />

3.	After that, To get the API key, click on **[Klaviyo API Console](https://www.klaviyo.com/account#api-keys-tab)** or   
**To Get the API Key**
-	Login your [Klaviyo dashboard](https://www.klaviyo.com/dashboard/) 
-	Go to Settings and click on API Keys.
-	Click on Create Private API key.
-	Copy the Private API Key and paste into API Key field of your authorization form.
-	Finally, click `Authorize` button.

 <Image src="actions/klaviyo/create-api" alt="Klaviyo" w="1020" h="350" />
 <Image src="actions/klaviyo/authorize" alt="Klaviyo" w="1020" h="350" />

4. Click on **`Next`** to set up the other integration settings ahead.

5. After completing the authorization process, Select the List from the drop-down option. You have to create a list before on your *Klaviyo* account. If you don't see any list. Click on the `refresh` button.

 <Image src="actions/klaviyo/klaviyo-list" alt="Klaviyo" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary form fields with the related Klaviyo field. Here **Email** field is the default for Klaviyo. Now, click on the `Next` button to move forward for further settings.
 <Image src="actions/klaviyo/klaviyo-field-map" alt="Klaviyo" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/klaviyo/finish-save" alt="Klaviyo" w="1020" h="350" />

## Integration Setting

1. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/klaviyo/klaviyo-timeline" alt="Klaviyo" w="1020" h="350" />
 <Image src="actions/klaviyo/klaviyo-log" alt="Klaviyo" w="1020" h="350" />

2. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/klaviyo/klaviyo-option" alt="Klaviyo" w="1020" h="350" />
