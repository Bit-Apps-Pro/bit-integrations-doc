> **Note: Mail Mint is available in the free and pro versions.**

***To integrate with Mail Mint follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Mail Mint** from the actions pages
 <Image src="actions/mail-mint/select-mail-mint" alt="Mail Mint" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Mail Mint'.
 <Image src="actions/mail-mint/integration-name" alt="Mail Mint" w="1020" h="350" />

3.	Click on the `Connect button`. 

 <Image src="actions/mail-mint/connect" alt="Mail Mint" w="1020" h="350" />

:::info
 **Note: Before setting up the integration, you must confirm that the Mail Mint plugin is activated on your site. Otherwise, the integration will not work.**
:::

4. Click on **`Next`** to set up the other integration settings ahead.

 <Image src="actions/mail-mint/next" alt="Mail Mint" w="1020" h="350" />


## Integration Settings:

5. Select the Add Contact action from the drop-down option.
 <Image src="actions/mail-mint/select-action" alt="Mail Mint" w="1020" h="350" />


5. After that, Select the ***Mail Main list*** from the drop-down option that you have created before on your Mail Mint account.
 <Image src="actions/mail-mint/select-list" alt="Mail Mint" w="1020" h="350" />

6. You can add a Tag [ also multiple tags] for the Mail Mint contacts that you have created before on your Mail Mint account.
 <Image src="actions/mail-mint/select-tag" alt="Mail Mint" w="1020" h="350" />

7. Select the contact status the selected status will added to the contact that will created through the integration.
 <Image src="actions/mail-mint/select-list" alt="Mail Mint" w="1020" h="350" />

8. Then, you have to map the form fields with ***Mail Mint*** fields. By default, the field is **Email**. You can select multiple fields from map field settings.  Then click on `Next` button after mapping the desired  field

 <Image src="actions/mail-mint/field-mapping" alt="Mail Mint" w="1020" h="350" />

9. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/mail-mint/finish-save" alt="Mail Mint" w="1020" h="350" />

10. Bit Integrations has an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/mail-mint/timeline" alt="Mail Mint" w="1020" h="350" />
 <Image src="actions/mail-mint/integration-log" alt="Mail Mint" w="1020" h="350" />

11. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/mail-mint/edit-clone-delete" alt="Mail Mint" w="1020" h="350" />
