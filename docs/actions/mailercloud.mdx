> **Note: Mailercloud is available in the free and pro versions.**

***To integrate with Mailercloud follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Mailercloud** from the actions pages
 <Image src="actions/mailercloud/search-and-select" alt="Mailercloud" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Mailercloud'.
 <Image src="actions/mailercloud/default-integration-name" alt="Mailercloud" w="1020" h="350" />

3.	After that, you have to copy the API key from your Mailercloud account & paste the API key on the field. To get the API key, click on the link or    
**To get Mailercloud API key**
-	First login your Mailercloud account
-	Go to Account > Integrations > API Integrations
-	If you haven't created an API first. Then create a new API.
- Copy the API Key and paste into API Key field of your authorization form. And Click on **Authorize** button
 <Image src="actions/mailercloud/click-here" alt="Mailercloud" w="1020" h="350" />
 <Image src="actions/mailercloud/visit-api-console" alt="Mailercloud" w="1020" h="350" />
 <Image src="actions/mailercloud/authorize" alt="Mailercloud" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/mailercloud/authorize-after" alt="Mailercloud" w="1020" h="350" />

5. After completing the authorization process, Select the `list` and `content Type` of your choices from the drop-down options. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/mailercloud/select-list-tag" alt="Mailercloud" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related Mailercloud field. By default, **`Email`** field of Mailercloud will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/mailercloud/mailercloud-field-map" alt="Mailercloud" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/mailercloud/finish-save" alt="Mailercloud" w="1020" h="350" />


## Actions 
8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/mailercloud/mailercloud-timeline" alt="Mailercloud" w="1020" h="350" />
 <Image src="actions/mailercloud/mailercloud-log" alt="Mailercloud" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/mailercloud/mailercloud-option" alt="Mailercloud" w="1020" h="350" />
