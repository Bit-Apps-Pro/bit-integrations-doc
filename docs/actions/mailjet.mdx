> **Note: Mailjet is only available on the pro version.**

***To integrate with Mailjet follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Mailjet** from the actions pages
 <Image src="actions/mailjet/search-and-select" alt="Mailjet" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Mailjet'.
 <Image src="actions/mailjet/default-integration-name" alt="Mailjet" w="1020" h="350" />

**To get Mailjet client id and client secret**
-	First login your Mailjet account
-	Go to setting > advanced setting > developer's corner > API Keys
-	Create a NEW API Key or copy the client id and client secret from the previously created API key
-	Then paste into client id and client secret field of your authorization form. And Click on **Authorize** button

 <Image src="actions/mailjet/click-here" alt="Mailjet" w="1020" h="350" />
 <Image src="actions/mailjet/authorize" alt="Mailjet" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/mailjet/authorize-after" alt="Mailjet" w="1020" h="350" />

5.	After completing the authorization process, Select the `List` of your choices from the drop-down options, and you can select multiple lists. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/mailjet/select-lists" alt="Mailjet" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related Mailjet field. By default, **`Email`** field of Mailjet will be available. You can also send custom values by mapping your required field and custom field. If you don't see any fields and missing. Click on the `refresh` button.

 <Image src="actions/mailjet/mailjet-field-map" alt="Mailjet" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/mailjet/finish-save" alt="Mailjet" w="1020" h="350" />


## Actions 
- **Is excluded from campaigns:** Indicates whether the contact is added to the exclusion list for campaigns or not. An excluded contact will not be receiving any marketing emails.  
 <Image src="actions/mailjet/exclude-campaigns" alt="Mailjet" w="1020" h="350" />


8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/mailjet/mailjet-timeline" alt="Mailjet" w="1020" h="350" />
 <Image src="actions/mailjet/mailjet-log" alt="Mailjet" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/mailjet/mailjet-option" alt="Mailjet" w="1020" h="350" />
