> **Note: Mailrelay is available in the free and pro versions.**

***To integrate with Mailrelay follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Mailrelay** from the actions pages
 <Image src="actions/mailrelay/search-and-select" alt="Mailrelay" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Mailrelay'.
 <Image src="actions/mailrelay/default-integration-name" alt="Mailrelay" w="1020" h="350" />

3.	After that, you have to copy the API Token from your Mailrelay account & paste the API Token on the field.     
**To get Mailrelay API Token**
- Input your domain name without toplevel domain in your domain field
- Then click on MailRelay API Token
- Click on Add to create a token 
- Copy the API key and paste into API Key field of your authorization form. And Click on **Authorize** button

 <Image src="actions/mailrelay/click-here" alt="Mailrelay" w="1020" h="350" />
 <Image src="actions/mailrelay/visit-api-console" alt="Mailrelay" w="1020" h="350" />
 <Image src="actions/mailrelay/authorize" alt="Mailrelay" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/mailrelay/authorize-after" alt="Mailrelay" w="1020" h="350" />

5.	After completing the authorization process, Select the `Subscriber Status ` of your choices from the drop-down options.
 <Image src="actions/mailrelay/select-status" alt="Mailrelay" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related Mailrelay field. By default, **`Email`** field of Mailrelay will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/mailrelay/mailrelay-field-map" alt="Mailrelay" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/mailrelay/finish-save" alt="Mailrelay" w="1020" h="350" />


## Actions 
- **Add Groups:** By enabling Add Group Action, you can organize your subscriber on the MailRelay account. Now select the group from the dropdown option, which you have already created on your MailRelay account.
 <Image src="actions/mailrelay/add-groups" alt="Mailrelay" w="1020" h="350" />

- **Update Lead:** You can also update the submitted data by enabling the action called Update Mailrelay. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address.
 <Image src="actions/mailrelay/update-lead" alt="Mailrelay" w="1020" h="350" />


8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/mailrelay/mailrelay-timeline" alt="Mailrelay" w="1020" h="350" />
 <Image src="actions/mailrelay/mailrelay-log" alt="Mailrelay" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/mailrelay/mailrelay-option" alt="Mailrelay" w="1020" h="350" />
