> **Note: Memberpress is available in the free and pro versions.**

***To integrate with Memberpress follow, the below steps.***

## Process of Authorization and Integration


1.	Search & Select **Memberpress** from the actions pages
 <Image src="actions/memberpress/select-action" alt="Memberpress" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken **Memberpress**.

3. Now, click on the `connect` button to complete authorization settings.

 <Image src="actions/memberpress/connect-button" alt="Memberpress" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.

 <Image src="actions/memberpress/authorize" alt="Memberpress" w="1020" h="350" />

## Actions List
### Add the user to a membership 
If you select this option from the drop-down. When users submit/complete the form/task, A new user will be added to your selected membership, or an error will occur if the user already exists in your group. You can see errors in the timeline. If you don't see any group. Click on the refresh button.

1.	After the authorization process, select the Action as **Add the user to a membership**. then select Membership. If you don't see any levels. Click on the `refresh` button.
  
 <Image src="actions/memberpress/select-dropdown-add-user" alt="Memberpress" w="1020" h="350" />

2.	You will find the Map Field option. You have to map all the necessary fields with the related Memberpress field. You can also send custom values by mapping your required field. Now, click on the `Next` button to move forward for further settings.
 <Image src="actions/memberpress/memberpress-fields-map" alt="Memberpress" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/memberpress/memberpress-finish-save" alt="Memberpress" w="1020" h="350" />

### Remove the user from a membership
If you select this option from the drop-down. When users submit/complete the form/task, Remove the user from your selected membership. If you don't see any group. Click on the refresh button.

## Actions
**Add Status:** Status is mandatory in Memberpress. You can add status (`Complete`, `Pending`, `Failed`, `Refunded`) from the `add status`.
 <Image src="actions/memberpress/memberpress-add-status" alt="Memberpress" w="1020" h="350" />

**Add Type:** Type is mandatory in Memberpress You can add a type (`Offline` `Payment`, `Manual`) from the Add Type.
 <Image src="actions/memberpress/memberpress-add-type" alt="Memberpress" w="1020" h="350" />

## Timeline

1. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/memberpress/memberpress-timeline" alt="Memberpress" w="1020" h="350" />
 <Image src="actions/memberpress/memberpress-timeline-log" alt="Memberpress" w="1020" h="350" />


2. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.
