> **Note: Moosend is only available on the pro version.**

***To integrate with Moosend follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Moosend** from the actions pages
 <Image src="actions/moosend/search-and-select" alt="Moosend" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Moosend'.
 <Image src="actions/moosend/default-integration-name" alt="Moosend" w="1020" h="350" />

3.	After that, you have to copy the API key from your Moosend account & paste the API key on the field.   
**To get Moosend API key**
-	First login your Moosend account
-	Goto setting and click on API.
- Copy the Token and paste into API Key field of your authorization form. And Click on **Authorize** button
 <Image src="actions/moosend/visit-api-console" alt="Moosend" w="1020" h="350" />
 <Image src="actions/moosend/authorize" alt="Moosend" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/moosend/authorize-after" alt="Moosend" w="1020" h="350" />

### Integration Settings for Subscribe

1.	After completing the authorization process, Select the `method` and `mailing list` of your choices from the drop-down options. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/moosend/select-method-mailing-list" alt="Moosend" w="1020" h="350" />

2.	You will find the Map Field option. You have to map all the necessary fields with the related *Moosend* field. By default, *Email* field of Moosend will be available. You can also send custom values by mapping your required field. Then click on `next` button.
 <Image src="actions/moosend/moosend-field-map" alt="Moosend" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/moosend/finish-save" alt="Moosend" w="1020" h="350" />
