> **Note: OmniSend is available in the free and pro versions.**

***To integrate with OmniSend follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **OmniSend** from the actions pages
 <Image src="actions/omni-send/search-and-select" alt="OmniSend" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'OmniSend'.
 <Image src="actions/omni-send/default-integration-name" alt="OmniSend" w="1020" h="350" />

3.	After that, you have to copy the client id and client secret from your OmniSend account & paste the client id and client secret on the field.     
**To get OmniSend client id and client secret**
-	Click on [OmniSend API account](https://app.omnisend.com/o/my-account/integrations/api-keys)
-	Create a NEW API Key or copy the api token from the previously created API key
-	Then paste into API Token field of your authorization form. And Click on “Authorize” button
 <Image src="actions/omni-send/click-here" alt="OmniSend" w="1020" h="350" />
 <Image src="actions/omni-send/visit-api-console" alt="OmniSend" w="1020" h="350" />
 <Image src="actions/omni-send/authorize" alt="OmniSend" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/omni-send/authorize-after" alt="OmniSend" w="1020" h="350" />

## Channels
### Email

7.	After completing the authorization process, Select the `Channels` and `Status` of your choices from the drop-down options. You can select multiple channels from the drop-down options. 
 <Image src="actions/omni-send/select-status" alt="OmniSend" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related OmniSend field. By default, **`Email`** field of OmniSend will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/omni-send/omni-send-field-map" alt="OmniSend" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/omni-send/finish-save" alt="OmniSend" w="1020" h="350" />

### SMS

7.	After completing the authorization process, Select the `Channels` and `Status` of your choices from the drop-down options. You can select multiple channels from the drop-down options. 
 <Image src="actions/omni-send/select-status-sms" alt="OmniSend" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related OmniSend field. By default, **`Phone number`** field of OmniSend will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/omni-send/omni-send-field-map-sms" alt="OmniSend" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/omni-send/finish-save" alt="OmniSend" w="1020" h="350" />



8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/omni-send/omni-send-timeline" alt="OmniSend" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/omni-send/omni-send-option" alt="OmniSend" w="1020" h="350" />
