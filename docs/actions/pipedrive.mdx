> **Note: PipeDrive is available in the free and pro versions.**

Pipedrive integrations via Bit Integrations is a seamless way to connect your PipeDrive account with other business tools and applications. With a wide range of integrations available. You can send data from your WordPress forms, Page/Theme Builder, WooCommerce, LMS, Membership, and other plugins to your PipeDrive account. For more details, refer to the integration guide provided.

## Process of Authorization and Integration

1.	Search & Select **PipeDrive** from the actions pages
 <Image src="actions/pipe-drive/select-pipe-drive" alt="PipeDrive" w="1020" h="350" />

2. **API Token:** To get API Token
-   Open your PipeDrive account.
-   Click on the Profile icon then Navigate to Personal preferences
-   Navigate to API

 <Image src="actions/pipe-drive/api-token" alt="PipeDrive" w="1020" h="350" />

3. After that, you have to copy API Token from your PipeDrive account & paste them into the integration setting. And Click on the `Authorize` button. After successfully authorized done. Click on `Next` to set up the other integration settings ahead.
 <Image src="actions/pipe-drive/authorized" alt="PipeDrive" w="1020" h="350" />

4.	**Module:** Choose the desired module from the options in the drop-down list.
 <Image src="actions/pipe-drive/select-module" alt="PipeDrive" w="1020" h="350" />

5.	**Organization:** Select the organization under the respective module. When you select an organization under the respective module in Pipedrive CRM, you will be able to view all the people related to that organization in the selected Organization view.

 <Image src="actions/pipe-drive/select-organization" alt="PipeDrive" w="1020" h="350" />

6.	**People:** Select the people from the drop-down list which you have already created in your Pipedrive CRM.
 <Image src="actions/pipe-drive/select-person" alt="PipeDrive" w="1020" h="350" />



7.	You will find the Map Field option. You have to map all the necessary fields with the related Pipedrive field. After mapping the desired fields, click on the `Next` button.
 <Image src="actions/pipe-drive/fieldmapping" alt="PipeDrive" w="1020" h="350" />

### Related list 

You can send data to the Pipedrive-related list from Trigger. Select the required related list module and layout from the drop-down provided. Then map the mandatory field. To map more fields, click on the (+).
 <Image src="actions/pipe-drive/releated-list" alt="PipeDrive" w="1020" h="350" />

### Actions

1. **Owners:** This option allows you can set a record owner of the sent record.
 <Image src="actions/pipe-drive/add-owners" alt="PipeDrive" w="1020" h="350" />

2. **Labels:** This option allows you to add the label of the record to your Pipedrive.
 <Image src="actions/pipe-drive/add-labels" alt="PipeDrive" w="1020" h="350" />

3. **Currency:** This option allows you to set the currency to the sent record
 <Image src="actions/pipe-drive/add-currency" alt="PipeDrive" w="1020" h="350" />

4. **Visible To:** This option allows you to control the visibility of the record that will be sent to the integration.
 <Image src="actions/pipe-drive/visible-to" alt="PipeDrive" w="1020" h="350" />

8. Now save the integration by clicking the Finish & Save Button

 <Image src="actions/pipe-drive/finish-save" alt="PipeDrive" w="1020" h="350" />

9. Bit Integrations is an option in every integration called **`Timeline`** where you can check integration success or error status.
 <Image src="actions/pipe-drive/timeline" alt="PipeDrive" w="1020" h="350" />
 <Image src="actions/pipe-drive/integration-log" alt="PipeDrive" w="1020" h="350" />

10. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/pipe-drive/edit-clone-delete" alt="PipeDrive" w="1020" h="350" />
