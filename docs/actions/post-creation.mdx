***To integrate with Post Creation follow, the below steps.***


## Process of Authorization and Integration


1.	Search & Select **Post Creation** from the actions pages
 <Image src="actions/post-creation/search-and-select" alt="Post Creation" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Post Creation'.
 <Image src="actions/post-creation/default-integration-name" alt="Post Creation" w="1020" h="350" />

3. After navigate to the integration page, select the ***Post Type*** you have created for the integration. For example, we have selected `Photo Album` as custom post type. 

 <Image src="actions/post-creation/select-post-type" alt="Post Creation" w="1020" h="350" />

4. Now, select **`Post Status`**, **`Author`** & **`Comment Status`** according to the requirement.
 <Image src="actions/post-creation/status-author-comment" alt="Post Creation" w="1020" h="350" />

5. For the field mapping option, map the form fields with the general/default **post fields**. 
 <Image src="actions/post-creation/post-fields-mapping" alt="Post Creation" w="1020" h="350" />

6. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/post-creation/next-page" alt="Post Creation" w="1020" h="350" />

7.	For the field mapping option, you need to configure two types of mapping of two popular plugins (**`ACF`**, **`Meta Box`**) as needed -
    -	Then map the form fields with the related **ACF** or **Meta Box** fields.
    -	Lastly map the file upload field.
 <Image src="actions/post-creation/acf-field-mapping" alt="Post Creation" w="1020" h="350" />
 <Image src="actions/post-creation/meta-box-field-mapping" alt="Post Creation" w="1020" h="350" />

8.	Click on the **`Next`** button to move forward for further settings.
 <Image src="actions/post-creation/move-to-forward" alt="Post Creation" w="1020" h="350" />

9.	Now save the integration by clicking the **`Finish & Save`** Button.
 <Image src="actions/post-creation/finish-and-save" alt="Post Creation" w="1020" h="350" />

10. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/post-creation/post-creation-timeline" alt="Post Creation" w="1020" h="350" />
 <Image src="actions/post-creation/post-creation-log" alt="Post Creation" w="1020" h="350" />

11. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/post-creation/post-creation-option" alt="Post Creation" w="1020" h="350" />
