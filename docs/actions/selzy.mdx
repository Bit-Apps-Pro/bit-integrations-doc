> **Note: Selzy is only available on the pro version.**

***To integrate with Selzy follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Selzy** from the actions pages
 <Image src="actions/selzy/search-and-select" alt="Selzy" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Selzy'.
 <Image src="actions/selzy/default-integration-name" alt="Selzy" w="1020" h="350" />

3.	After that, you have to copy the API key from your Selzy account & paste the API key on the field. To get the API key, click on the link or    
**To get Selzy API key**
- Goto Settings and click on Integration and API.
- API access section API key click show full.
- Enter your password and click send
- Copy the API Key and paste into API Key field of your authorization form. And Click on **Authorize** button
 <Image src="actions/selzy/click-here" alt="Selzy" w="1020" h="350" />
 <Image src="actions/selzy/visit-api-console" alt="Selzy" w="1020" h="350" />
 <Image src="actions/selzy/authorize" alt="Selzy" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.

5. After completing the authorization process, Select the action type from the drop-down options (`Subscribe`, `Unsubscribe`).

6. Then select the **List** from the drop-down option and the **Tag** from your choices dropdown option; you can choices multiple lists and tags.  If you don't see any list and tag. Click on the `refresh` button.
 <Image src="actions/selzy/select-list-tag" alt="Selzy" w="1020" h="350" />

7.	You will find the Map Field option. You have to map all the necessary fields with the related Selzy field. By default, **`“Email”`** field of Selzy will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/selzy/selzy-field-map" alt="Selzy" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/selzy/finish-save" alt="Selzy" w="1020" h="350" />

8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/selzy/selzy-timeline" alt="Selzy" w="1020" h="350" />
 <Image src="actions/selzy/selzy-log" alt="Selzy" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/selzy/selzy-option" alt="Selzy" w="1020" h="350" />
