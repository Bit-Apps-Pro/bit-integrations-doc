> **Note: Sendfox is only available on the pro version.**

***To integrate with Sendfox follow, the below steps.***


## Process of Authorization and Integration
1.	Search & Select **Sendfox** from the actions pages
 <Image src="actions/sendfox/select-action" alt="Sendfox" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken Sendfox integration.
 <Image src="actions/sendfox/integration-name" alt="Sendfox" w="1020" h="350" />

3.	After that, you have to copy the Sendfox Access Token from your Sendfox account & paste the key on the field.    
**To get the Sendfox Access token,** 
- Click on `Sendfox Access Token`
- Click on `Create New Toke`n and Set proper name
- Then click on `create` button
- Now copy the Access Token, paste it into the integrations dashboard. And Click on` “Authorize”` button.

 <Image src="actions/sendfox/click-access-token" alt="Sendfox" w="1020" h="350" />
 <Image src="actions/sendfox/access-token" alt="Sendfox" w="1020" h="350" />
 <Image src="actions/sendfox/authorize" alt="Sendfox" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.

 <Image src="actions/sendfox/after-authorize" alt="Sendfox" w="1020" h="350" />


### Integration Settings for Create List
 1.	After the authorization process, select the Action **Create List** to create a Subscribers list.
 <Image src="actions/sendfox/select-action-create-list" alt="Sendfox" w="1020" h="350" />

2. When you select the Action, you will see the Map Fields option where a field, ***List Name***, is mapped by default. You can map the fields according to your choice for the list Name. And click on the `Next` button to move forward for further settings.
 <Image src="actions/sendfox/create-list-map-fields" alt="Sendfox" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/sendfox/finish-save" alt="Sendfox" w="1020" h="350" />

### Integration Settings for Create Contact
1.	After the authorization process, select the Action as **Create Contact** to create a Contact under List, then select **List**. You can add multiple List. If you don't see any List. Click on the refresh button.
 <Image src="actions/sendfox/select-action-create-contact-and-list" alt="Sendfox" w="1020" h="350" />

2. When you select the **Actions** and **List**, you will see the Map Fields option where the Sendfox email field, is mapped by default. You can map fields according to your choice for the *Sendfox contact fields*. And click on the `Next` button to move forward for further settings.
 <Image src="actions/sendfox/contact-list-fields-map" alt="Sendfox" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button
 <Image src="actions/sendfox/finish-save" alt="Sendfox" w="1020" h="350" />



### Integration Settings for Unsubscribe Contact
 1.	After the authorization process, select the Action **Unsubscribe Contact** to Unsubscribe a Subscribers list, 
 <Image src="actions/sendfox/select-action-unsubscribe-contact" alt="Sendfox" w="1020" h="350" />

2. When you select the Action, you will see the Map Fields option where a field, the *Sendfox Email field*, is mapped by default. You can map the fields. And click on the `Next` button to move forward for further settings.
 <Image src="actions/sendfox/unsubscribe-contact-map-fields" alt="Sendfox" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/sendfox/finish-save" alt="Sendfox" w="1020" h="350" />
