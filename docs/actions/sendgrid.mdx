> **Note: SendGrid is available in the free and pro versions.**

***To integrate with SendGrid follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **SendGrid** from the actions pages
 <Image src="actions/sendgrid/search-and-select" alt="SendGrid" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'SendGrid'.
 <Image src="actions/sendgrid/default-integration-name" alt="SendGrid" w="1020" h="350" />

3.	After that, you have to copy the API Token from your SendGrid account & paste the key on the field.    
**To get the API Token**, 
-	Click on [SendGrid API Token](https://app.sendgrid.com/settings/api_keys) or go to your SendGrid dashboard 
-	Setting -> API Keys -> Create Api Key
-	Put your API key Name and click on Create and view  button
-	Now copy the API Token and paste it into the integrationâ€™s dashboard. And Click on **Authorize** button.
 <Image src="actions/sendgrid/click-here" alt="SendGrid" w="1020" h="350" />
 <Image src="actions/sendgrid/visit-api-console" alt="SendGrid" w="1020" h="350" />
 <Image src="actions/sendgrid/authorize" alt="SendGrid" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/sendgrid/authorize-after" alt="SendGrid" w="1020" h="350" />

5.	After completing the authorization process, You will find the Map Field option. You have to map all the necessary fields with the related SendGrid field. By default, **Email** field of SendGrid will be available. You can also send custom values by mapping your required field. If you don't see any fields. Click on the `refresh` button.

 <Image src="actions/sendgrid/sendgrid-field-map" alt="SendGrid" w="1020" h="350" />

6. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/sendgrid/finish-save" alt="SendGrid" w="1020" h="350" />


## Actions 
- **Select lists:** You can enable the action Select Lists Field to add the contacts of the lists. In addition, you can select multiple list from the Drop-Down list.  
 <Image src="actions/sendgrid/select-lists" alt="SendGrid" w="1020" h="350" />


7. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/sendgrid/sendgrid-timeline" alt="SendGrid" w="1020" h="350" />
 <Image src="actions/sendgrid/sendgrid-log" alt="SendGrid" w="1020" h="350" />

8. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/sendgrid/sendgrid-option" alt="SendGrid" w="1020" h="350" />
