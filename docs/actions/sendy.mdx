> **Note: Sendy is available in the free and pro versions.**

***To integrate with Sendy follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Sendy** from the actions pages
 <Image src="actions/sendy/search-and-select" alt="Sendy" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Sendy'.
 <Image src="actions/sendy/default-integration-name" alt="Sendy" w="1020" h="350" />

3.	After that, you have to copy the API key from your Sendy account & paste the API key on the field.     
**To get Sendy API key**
-	First login your Sendy account
-	Go to Account > Setting 
-   Copy the API Key and paste into API Key field of your authorization form. And Click on **Authorize** button
 <Image src="actions/sendy/click-here" alt="Sendy" w="1020" h="350" />
 <Image src="actions/sendy/visit-api-console" alt="Sendy" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/sendy/authorize-after" alt="Sendy" w="1020" h="350" />

5.	After completing the authorization process, Select the `Brand` and `List` of your choices from the drop-down options. If you don't see any lists. Click on the `refresh` button.
 <Image src="actions/sendy/select-brand-list" alt="Sendy" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related Sendy field. By default, **`Email`** field of Sendy will be available. You can also send custom values by mapping your required field. then, click on the **`Next`** button to move forward for further settings.

 <Image src="actions/sendy/sendy-field-map" alt="Sendy" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/sendy/finish-save" alt="Sendy" w="1020" h="350" />


## Actions 
8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/sendy/sendy-timeline" alt="Sendy" w="1020" h="350" />
 <Image src="actions/sendy/sendy-log" alt="Sendy" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/sendy/sendy-option" alt="Sendy" w="1020" h="350" />
