> **Note: SureCart is only available on the pro version.**

***To integrate with SureCart follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **SureCart** from the actions pages
 <Image src="actions/surecart/search-and-select" alt="SureCart" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'SureCart'.
 <Image src="actions/surecart/default-integration-name" alt="SureCart" w="1020" h="350" />

3.	After that, you have to copy the API key from your SureCart account & paste the key on the field.   
To get the API key,   
-	Click on **`SureCart`**
-	go to your **SureCart developer** tab
-	Click on **New API Token** to create a new key
-	Or click on the previous API key
-	Copy the API key and paste into API Key field of your authorization form. And Click on `Authorize` button.

 <Image src="actions/surecart/click-api" alt="SureCart" w="1020" h="350" />
 <Image src="actions/surecart/visit-api-console" alt="SureCart" w="1020" h="350" />
 <Image src="actions/surecart/surecart-account-api-copy" alt="SureCart" w="1020" h="350" />
 <Image src="actions/surecart/authorize" alt="SureCart" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/surecart/authorize-after" alt="SureCart" w="1020" h="350" />

5.	After completing the authorization process, Select `Actions(Create Customer)` from the drop-down options. 
 <Image src="actions/surecart/select-action" alt="SureCart" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related SureCart field. By default, **`“Customer First Name”`**, **`“Email”`** field of SureCart will be available. You can also send custom values by mapping your required field. After mapping the desired fields, click on Next button.
 <Image src="actions/surecart/surecart-field-map" alt="SureCart" w="1020" h="350" />
<details>
      <summary>
       <b>✅ Available SureCart mapping fields</b>
      </summary>
      <div>
         <ul>
           <li>Customer First Name</li>
           <li>Customer Last Name</li>
           <li>Customer Email</li>
           <li>Customer Phone</li>
         </ul>
      </div>
</details>

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/surecart/finish-save" alt="SureCart" w="1020" h="350" />

8. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/surecart/surecart-timeline" alt="SureCart" w="1020" h="350" />
 <Image src="actions/surecart/surecart-log" alt="SureCart" w="1020" h="350" />

9. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/surecart/surecart-option" alt="SureCart" w="1020" h="350" />
