***To integrate with Trello follow, the below steps.***


## Process of Authorization and Integration
> **Note: Trello is available on the free and pro version.**

1.	Search & Select **Trello** from the actions pages
 <Image src="actions/trello/search-and-select" alt="Trello" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken 'Trello'.
 <Image src="actions/trello/default-integration-name" alt="Trello" w="1020" h="350" />

3.	After that, to get the API key, click on `Trello API Console`  
    **Setup Process**
    1.	Copy **Authorized Redirect URIs**
    2.	Scroll down and go to **'Allowed Origins'** section, paste **‘New Allowed Origin’** field.
    3.	Click on **Submit** button

    <Image src="actions/trello/allowed-origins" alt="Trello" w="1020" h="350" />

    4.	Now copy the **API key** and paste it into the integration’s dashboard. And Click on `“Authorize”` button.

    <Image src="actions/trello/developer-api-key" alt="Trello" w="1020" h="350" />
    <Image src="actions/trello/authorize" alt="Trello" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.

5.	After completing the authorization process, Select the Boards from the drop-down option. You have to create a Boards before on your trello account. And select the list form the drop-down option.
 <Image src="actions/trello/select-borad-and-list" alt="Trello" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary form fields with the related Trello Board field. Here ‘Name ‘field is the default for Trello Board. 

 <Image src="actions/trello/trello-field-map" alt="Trello" w="1020" h="350" />

## Actions
1.	If you enable this option. You can display your message at the top or bottom of the list as you wish. Now, click on the `Next` button to move forward for further settings.
 <Image src="actions/trello/add-position" alt="Trello" w="1020" h="350" />

2. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/trello/finish-save" alt="Trello" w="1020" h="350" />

3. We have an option in every integration **`Timeline`** where you can check integration success or error status.
 <Image src="actions/trello/trello-timeline" alt="Trello" w="1020" h="350" />
 <Image src="actions/trello/trello-log" alt="Trello" w="1020" h="350" />

4. Also, you can **`Edit`** **`Clone`** **`Delete`** the integration.

 <Image src="actions/trello/trello-option" alt="Trello" w="1020" h="350" />
