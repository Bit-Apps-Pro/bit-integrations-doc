> **Note: Tutor LMS is available in the free and pro versions.**

***To integrate with Tutor LMS follow, the below steps.***


## Process of Authorization and Integration

1. Search & Select **Tutor LMS** from the actions pages
 <Image src="actions/tutor-lms/select-tutor-lms" alt="Tutor-LMS" w="1020" h="350" />
 
2. **Integration Name:** Set the proper integration name. Default it's taken “Tutor LMS ”.
 <Image src="actions/tutor-lms/integration-name" alt="Tutor-LMS" w="1020" h="350" />

3.	Now click on the **Connect** Button to connect with Tutor LMS; after successfully connect, click on the **Next** button.
 <Image src="actions/tutor-lms/authorize" alt="Tutor-LMS" w="1020" h="350" />
 <Image src="actions/tutor-lms/next-button" alt="Tutor-LMS" w="1020" h="350" />



4. Select the Action (`Enroll the user in a course`, `Unenroll user from a course`, `Mark a course complete for the user`, `Mark a lesson complete for the user`, `Reset the user progress in a course`) from the Drop-Down list, And Select the Course/ Lessons from the Drop-Down list that you want to integrate with Tutor LMS . Then click on the **Next** button.
 <Image src="actions/tutor-lms/select-action-and-course" alt="Tutor-LMS" w="1020" h="350" />

5. Now save the integration by clicking the "**Finish & Save.**" Button
 <Image src="actions/tutor-lms/finis-and-save" alt="Tutor-LMS" w="1020" h="350" />
