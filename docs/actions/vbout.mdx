> **Note: Vbout is available in the free and pro versions.**

***To integrate with Vbout follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Vbout** from the actions pages
 <Image src="actions/vbout/select-action-vbout" alt="Vbout" w="1020" h="350" />

2. **Integration Name:** Set the proper integration name. Default it's taken **Vbout**.
 <Image src="actions/vbout/integration-name" alt="Vbout" w="1020" h="350" />

3.	After that, you have to copy the API Token from your Vbout account & paste the key on the field.    
To get the API Token,    
-	click on `Vbout API Token` or go to your **Vbout dashboard** 
-	click on Profile icon => Setting => Navigate to API Integrations
-	Now copy the Your Unique API User Key and paste it into the integration’s dashboard. And Click on **“Authorize”** button

 <Image src="actions/vbout/click-api-token" alt="Vbout" w="1020" h="350" />
 <Image src="actions/vbout/copy-api" alt="Vbout" w="1020" h="350" />
 <Image src="actions/vbout/authorize" alt="Vbout" w="1020" h="350" />


4. Click on **`Next`** to set up the other integration settings ahead.
 <Image src="actions/vbout/after-authorize" alt="Vbout" w="1020" h="350" />

5.	After completing the authorization process, select the Contacts list that you have created before on Vbout. By default, the name of the list is Sample list. Then elect the Contact Status from the drop-down option (Active, Unconfirmed).  If you don't see any Contacts list. Click on the `refresh button`.
 <Image src="actions/vbout/vbout-list-status" alt="Vbout" w="1020" h="350" />

6.	You will find the Map Field option. You have to map all the necessary fields with the related Vbout field. By default, **“Email”** field of Vbout will be available. You can also send custom values by mapping your required field. Now, click on the `Next` button to move forward for further settings. If you don't see any Vbout fields. Click on the `refresh button`.
 <Image src="actions/vbout/vbout-field-map" alt="Vbout" w="1020" h="350" />

7. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/vbout/finish-save" alt="Vbout" w="1020" h="350" />
 <Image src="actions/vbout/check-integration" alt="Vbout" w="1020" h="350" />

8. We have an option in every integration **"Timeline"** where you can check integration success or error status.
    <Image src="actions/vbout/integration-dashboard" alt="Vbout" w="1020" h="350" />
    <Image src="actions/vbout/timeline" alt="Vbout" w="1020" h="350" />

9.	Also, you can `Edit`, `Clone`, and, `Delete` the integration.
    <Image src="actions/vbout/edit-clone" alt="Vbout" w="1020" h="350" />
    
## Actions

- **Update Contact:** If the user already exists on your Vbout account, It will update the contacts with other field information based on the email address.
 <Image src="actions/vbout/update" alt="Vbout" w="1020" h="350" />
