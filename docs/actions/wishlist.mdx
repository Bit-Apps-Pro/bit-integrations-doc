> **Note: WishList is only available on the pro version.**

***To integrate with WishList follow, the below steps.***


## Process of Authorization and Integration


1.	Search & Select **WishList** from the actions pages
 <Image src="actions/wishlist/select-action" alt="WishList" w="1020" h="350" />

2. Integration Name: Set the proper integration name. Default it's taken WishList integration.
 <Image src="actions/wishlist/integration-name" alt="WishList" w="1020" h="350" />

3.	After that, you have to copy the **WishList Member URL** and **Password** from your WishList account & paste the key on the field. 
To get the WishList Member URL and Password, 
-	Go to WishList Member dashboard 
-	 Click on Advanced option and go to API. 
-	Now copy the API URL and Password, paste it into the integrations dashboard. And Click on **Authorize** button
 <Image src="actions/wishlist/api-and-site-url" alt="WishList" w="1020" h="350" />
 <Image src="actions/wishlist/authorize" alt="WishList" w="1020" h="350" />

4.	Click on `Next` to set up the other integration settings ahead.

 <Image src="actions/wishlist/after-authorize" alt="WishList" w="1020" h="350" />


### Integration Settings for Add Member under Level
1.	After the authorization process, select the `Action` as **Add Member under Level** to create a member under level, then select `Levels`. If you don't see any levels. Click on the refresh button. 
 <Image src="actions/wishlist/select-action-and-level" alt="WishList" w="1020" h="350" />

2.	When you select the Action and Levels, you will see the `Map Fields` option where two fields- **Email** & **Username**, are mapped by default. You can map multiple fields according to your choice for the **member Action**. And click on the `Next` button to move forward for further settings.
 <Image src="actions/wishlist/map-fields" alt="WishList" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/wishlist/finish-save" alt="WishList" w="1020" h="350" />

### Integration Settings for Add Level under Member
1.	After the authorization process, select the `Action` as **Add Level under Member** to create a Level under member, then select `Members`. If you don't see any Members. Click on the refresh button.
 <Image src="actions/wishlist/select-action-and-level-two" alt="WishList" w="1020" h="350" />

2. When you select the `Action` and `Levels`, you will see the Map Fields option where the one **level name**, is mapped by default. You can map fields according to your choice for the level name. And click on the `Next` button to move forward for further settings.
 <Image src="actions/wishlist/level-fields-map" alt="WishList" w="1020" h="350" />

3. Now save the integration by clicking the **`Finish & Save`** Button

 <Image src="actions/wishlist/finish-save-two" alt="WishList" w="1020" h="350" />
