> **Note: Zoho Bigin is available in the free and pro versions.**

***To integrate with Zoho Bigin follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Zoho Bigin** from the actions pages
 <Image src="actions/zoho-bigin/_select-zoho-bigin" alt="Zoho Bigin" w="1020" h="350" />

4.	Integration Name: Set the proper integration name Default it's taken “Zoho Bigin”.
 <Image src="actions/zoho-bigin/_integration-name" alt="Zoho Bigin" w="1020" h="350" />

5. Select a data center from the dropdown. Then click on the *Zoho API Console*. Navigate to **API Console-**> **Get Started**-> **Server-based Applications.** 
 <Image src="actions/zoho-bigin/_select-data-center" alt="Zoho Bigin" w="1020" h="350" />
 <Image src="actions/zoho-bigin/_get-started" alt="Zoho Bigin" w="1020" h="350" />
 <Image src="actions/zoho-bigin/_server-based-applications" alt="Zoho Bigin" w="1020" h="350" />


6.	Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click **CREATE** button
 <Image src="actions/zoho-bigin/_create-api" alt="Zoho Bigin" w="1020" h="350" />

 
7.	A Client ID and a Client Secret will be generated. Then go to the Client Secret tab
 <Image src="actions/zoho-bigin/_client-id-and-serect" alt="Zoho Bigin" w="1020" h="300" />

8.	Then copy the Client ID and Client Secret and paste it to the Trigger integration page. Then click on the **Authorize** button (wait for a few seconds). After the successful authorization click on the **Next** button
 <Image src="actions/zoho-bigin/_authorize" alt="Zoho Bigin" w="1020" h="300" />

9.	Select a **Module** from the drop-down list. Map all the mandatory Recruit fields to the corresponding fields in your form.
 <Image src="actions/zoho-bigin/_select-module" alt="Zoho Bigin" w="1020" h="300" />
 <Image src="actions/zoho-bigin/_field-mapping" alt="Zoho Bigin" w="1020" h="300" />

## Actions

1. **Workflow:** Select this option to trigger the Zoho Bigin workflow for the selected module. If you didn’t select this option Zoho Bigin workflow for the selected module doesn’t work.
 <Image src="actions/zoho-bigin/_workflow-actions" alt="Zoho Bigin" w="1020" h="350" />

2. **Attachment:** To Select this option you can send file upload field data to Zoho Bigin related list Attachments. You can also send file upload field data to Zoho Bigin via normal field mapping). Please select the CRM file upload field from the drop-down which you want to send data.
 <Image src="actions/zoho-bigin/_attachment-actions" alt="Zoho Bigin" w="1020" h="350" />


3. **Approval:** Select this option to automatically send form data to your Zoho Bigin approval list instead of adding them.
 <Image src="actions/zoho-bigin/_approval-actions" alt="Zoho Bigin" w="1020" h="350" />

4. **Add Note:** Select this action to add a note from Trigger to Zoho Bigin account automatically.
 <Image src="actions/zoho-bigin/_add-notes-actions" alt="Zoho Bigin" w="1020" h="350" />

5. **Upload Photo:** You can update any contact’s profile photo through our Trigger to your Zoho Bigin account. For that, hoover on the action called “Upload Photo“. After mapping the relevant fields, click the action ‘Upload Photo’ to upload a photo from Trigger to Zoho Bigin. Now click on ‘Save’. Whenever the user fills-up the form on Trigger, the image will be automatically uploaded as a profile photo to the Zoho Bigin account.
 <Image src="actions/zoho-bigin/_upload-photo-actions" alt="Zoho Bigin" w="1020" h="350" />

## Save Integration
1.	After giving all the required information, click on **Finish & Save**. Now your created **Zoho Bigin API** is ready for implementation.
