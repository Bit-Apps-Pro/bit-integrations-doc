> **Note: Zoho Desk is only available on the pro version.**

***To integrate with Zoho Desk follow, the below steps.***


## Process of Authorization and Integration

1.	Search & Select **Zoho Desk** from the actions pages
 <Image src="actions/zoho-desk/_select-zoho-desk" alt="Zoho Desk" w="1020" h="350" />

4.	Integration Name: Set the proper integration name Default it's taken “Zoho Desk”.
 <Image src="actions/zoho-desk/_integration-name" alt="Zoho Desk" w="1020" h="350" />

5. Select a data center from the dropdown. Then click on the *Zoho API Console*. Navigate to **API Console-**> **Get Started**-> **Server-based Applications.** 
 <Image src="actions/zoho-desk/_data-center" alt="Zoho Desk" w="1020" h="350" />
 <Image src="actions/zoho-desk/_get-started" alt="Zoho Desk" w="1020" h="350" />
 <Image src="actions/zoho-desk/_server-based-applications" alt="Zoho Desk" w="1020" h="350" />


6.	Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click **CREATE** button
 <Image src="actions/zoho-desk/_create-api" alt="Zoho Desk" w="1020" h="350" />

 
7.	A Client ID and a Client Secret will be generated. Then go to the Client Secret tab
 <Image src="actions/zoho-desk/_client-id-and-secret" alt="Zoho Desk" w="1020" h="300" />

8.	Then copy the Client ID and Client Secret and paste it to the Bit Integrations integration page. Then click on the **Authorize** button (wait for a few seconds). After the successful authorization click on the **Next** button
 <Image src="actions/zoho-desk/_authorize" alt="Zoho Desk" w="1020" h="300" />

9.	Select the required **Portal and Department** from the dropdown
 <Image src="actions/zoho-desk/_select-portal-and-department" alt="Zoho Desk" w="1020" h="300" /> 

10. When you fill-up the **Portal and Department** from the dropdown.You will find the **Map** Field option.You have to **Map** all the mandatory Desk fields to the corresponding fields in your form/Task. At last select the requires **Ticket Owner** from the **Actions.**
 <Image src="actions/zoho-desk/_select-portal-and-department" alt="Zoho Desk" w="1020" h="300" />
To map more form fields to Desk fields, click on the (+). The form fields that are relevant to the selected field would be listed in the drop-down under Bit Forms.


## Actions

1. **Ticket Owner:** Ticket Owner is mandatory in Zoho Desk. You have to select the Ticket Owner from actions
 <Image src="actions/zoho-desk/_titcket-owner-action" alt="Zoho Desk" w="1020" h="350" />

2. **Product Name:** You can product-wise assign ticket in Zoho Desk. For this, you have to select the Product Name from the actions.
 <Image src="actions/zoho-desk/_product-name-action" alt="Zoho Desk" w="1020" h="350" />


3. **Attachments:** To send an attachment to the Zoho Desk you have to select the required file upload field from actions.
 <Image src="actions/zoho-desk/_attachment-actions" alt="Zoho Desk" w="1020" h="350" />


## Save Integration
1.	After clicking on **Finish & Save**, the Zoho Desk integration is ready.
 <Image src="actions/zoho-desk/_finish-and-save" alt="Zoho Desk" w="1020" h="350" />
