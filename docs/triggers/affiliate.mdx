> **Note: Affiliate is available in the free and pro versions.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/affiliate/create-integration" alt="Affiliate" w="1020" h="350" />

2. Now search and select the **Affiliate** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/affiliate/affiliate" alt="Affiliate" w="1020" h="350" />

3. Select the task from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/affiliate/select-task" alt="Affiliate" w="1020" h="350" />


#### ✅ The available task in Affiliate

**👉 A new affiliate is approved:** This integration will be triggered when when a new affiliate is approved.    
*For example,* when a new affiliate is approved, you can use this task to send the affiliate user details to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**👉 A user becomes an affiliate:** This integration will be triggered when you add a user to an affiliate.    
*For example,* when a user become an affiliate, you can use this task to send the affiliate user and affiliate details to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**👉An affiliate makes a referral of a specific type:** This integration will be triggered when an affiliate create/add a referral of a specific type.You can set a specific type referral from drop-down.
   
*For example,* when an affiliate create/add a referral of a specific type, you can use this task to send the referral description and affiliate details to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**👉 An affiliates referral of a specific type is rejected Pro** This integration will be triggered when an affiliate create/add a referral of a specific type and the status is rejected    
*For example,* when an affiliate create/add a referral of a specific type and the status is rejected, you can use this task to send the referral description and affiliate details to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.

**👉 An affiliates referral of a specific type is paid Pro** This integration will be triggered when an affiliate create/add a referral of a specific type and the status is paid. You can set a specific type referral  from drop-down.   
*For example,*when an affiliate create/add a referral of a specific type and the status is paid, you can use this task to send the referral description and affiliate details to [Google Sheets](https://docs.bit-integrations.bitapps.pro/integrations/google-sheet), [Mailchimp](https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp), Email marketing tools, and other third-party platforms.


<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
