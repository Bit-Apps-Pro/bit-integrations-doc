> **Note: Memberpress is available in the free and pro versions.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="Memberpress" w="1020" h="350" />

2. Now select the **Memberpress** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/memberpress/select-trigger" alt="Memberpress" w="1020" h="350" />

3. Select the Task from the Drop-Down list, And Select the membership from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. Then click on the **Next** button.

   <Image src="trigger/memberpress/select-form-or-task-name" alt="Memberpress" w="1020" h="350"/>
   

#### ✅ The available task in Memberpress

      <summary>
       <b>A user purchases a one-time subscription membership</b>
