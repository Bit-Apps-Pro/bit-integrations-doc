> **Note: Paid Membership Pro is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="Paid Membership Pro" w="1020" h="350" />

2. Now select the **Paid Membership Pro** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/paid-membership-pro/select-trigger" alt="Paid Membership Pro" w="1020" h="350" />

3. Select the Task from the Drop-Down list, And Select the membership from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. Then click on the **Next** button.

   <Image src="trigger/paid-membership-pro/select-form-or-task-name" alt="Paid Membership Pro" w="1020" h="350"/>
