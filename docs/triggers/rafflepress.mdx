> **Note: Rafflepress is available on the free and pro version.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="Rafflepress" w="1020" h="350" />

2. Now select the **Rafflepress** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/rafflepress/select-trigger" alt="Rafflepress" w="1020" h="350" />

3. Select the form from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/rafflepress/select-form-or-task-name" alt="Rafflepress" w="1020" h="350"/>
   

#### ✅ The available task in Rafflepress

<details>
      <summary>
       <b>User login giveaway </b>
      </summary>
      <div>
        This integration will be triggered when the User login a specific giveaway. <br/> 
         <b>For example,</b>when the User login a specific giveaway, you can use this task to send the user and giveaway details to <a href="https://docs.bit-integrations.bitapps.pro/integrations/google-sheet">Google Sheets</a>, <a href="https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp">Mailchimp</a>, CRM, Email marketing tools, and other third-party platforms.<br/><br/>
         <b>✅ Available task fields</b>
         <ul>
          <li>Giveaway ID</li>
         <li>Giveaway Name</li>
         <li>Starts</li>
         <li>Ends</li>
         <li>Active</li>
         <li>Name</li>
         <li>First Name</li>
         <li>Last Name</li>
         <li>Email</li>
         <li>Prize Name</li>
         <li>Prize Description</li>
         <li>Prize Image</li>
         </ul>
      </div>
</details>




<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
