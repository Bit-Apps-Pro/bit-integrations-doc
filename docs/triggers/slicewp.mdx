> **Note: SliceWP is available in the free and pro versions.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="SliceWP" w="1020" h="350" />

2. Now select the **SliceWP** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/slicewp/select-trigger" alt="SliceWP" w="1020" h="350" />

3. Select the form from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/slicewp/select-form-or-task-name" alt="SliceWP" w="1020" h="350"/>
   

#### ✅ The available task in SliceWP

<details>
      <summary>
       <b>User becomes an affiliate</b>
      </summary>
      <div>
        This integration will be triggered when you add a user to an affiliate.<br/> 
         <b>For example,</b>when a user become an affiliate, you can use this task to send the affiliate user and affiliate details to <a href="https://docs.bit-integrations.bitapps.pro/integrations/google-sheet">Google Sheets</a>, <a href="https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp">Mailchimp</a>, CRM, Email marketing tools, and other third-party platforms.<br/><br/>
         <b>✅ Available task fields</b>
         <ul>
            <li>Affiliate ID</li>
            <li>User ID</li>
            <li>Payment Email</li>
            <li>Website URL</li>
            <li>Date Created</li>
            <li>Status</li>
            <li>First Name</li>
            <li>Last Name</li>
            <li>Nick Name</li>
            <li>Avatar URL</li>
            <li>Email</li>
         </ul>
      </div>
</details>

<details>
      <summary>
       <b>User earns a commission</b>
      </summary>
      <div>
        This integration will be triggered when the user earns a commission. You can set a specific sale or subscription or any.<br/> 
         <b>For example,</b>when a user earns a commission, you can use this task to send the affiliated person and earns commission details to <a href="https://docs.bit-integrations.bitapps.pro/integrations/google-sheet">Google Sheets</a>, <a href="https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp">Mailchimp</a>, CRM, Email marketing tools, and other third-party platforms.<br/><br/>
         <b>✅ Available task fields</b>
         <ul>
            <li>Commission ID</li>
            <li>Affiliate ID</li>
            <li>Date Created</li>
            <li>Amount</li>
            <li>Reference</li>
            <li>Origin</li>
            <li>Type</li>
            <li>Status</li>
            <li>Currency</li>
         </ul>
      </div>
</details>

<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
