> **Note: Solid Affiliate is available in the free and pro versions.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="Solid Affiliate" w="1020" h="350" />

2. Now select the **Solid Affiliate** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/solid-affiliate/select-trigger" alt="Solid Affiliate" w="1020" h="350" />

3. Select the form from the Drop-Down list that you want to integrate with Bit Integrations **Actions**. And click on the **“Next”** button.

   <Image src="trigger/solid-affiliate/select-form-or-task-name" alt="Solid Affiliate" w="1020" h="350"/>
   

#### ✅ The available task in Solid Affiliate

<details>
      <summary>
       <b> Created new affiliate</b>
      </summary>
      <div>
        This integration will be triggered when the created new affiliate is. <br/> 
         <b>For example,</b> When a new affiliate is created, you can use this task to send the affiliate person details to <a href="https://docs.bit-integrations.bitapps.pro/integrations/google-sheet">Google Sheets</a>, <a href="https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp">Mailchimp</a>, CRM, Email marketing tools, and other third-party platforms.<br/><br/>
         <b>✅ Available task fields</b>
         <ul>
           <li>User ID</li>
            <li>First Name</li>
            <li>Last Name</li>
            <li>Commission Type</li>
            <li>Commission Rate</li>
            <li>Payment Email</li>
            <li>Mailchimp User ID</li>
            <li>Affiliate Group ID</li>
            <li>Registration Notes</li>
            <li>Status</li>
            <li>Created At</li>
            <li>Updated At</li>
         </ul>
      </div>
</details>

<details>
      <summary>
       <b>Created new referral affiliate </b>
      </summary>
      <div>
        This integration will be triggered when the created new referral affiliate is.<br/> 
         <b>For example,</b> when a new affiliate is created, you can use this task to send the affiliate person id and order details to <a href="https://docs.bit-integrations.bitapps.pro/integrations/google-sheet">Google Sheets</a>, <a href="https://docs.bit-integrations.bitapps.pro/integrations/wpcf7-mailchimp">Mailchimp</a>, CRM, Email marketing tools, and other third-party platforms.<br/><br/>
         <b>✅ Available task fields</b>
         <ul>
            <li>Affiliate ID</li>
            <li>Order Amount</li>
            <li>Commission Amount</li>
            <li>Referral Source</li>
            <li>Visit ID</li>
            <li>Coupon ID</li>
            <li>Customer ID</li>
            <li>Referral Type</li>
            <li>Description</li>
            <li>Order Source</li>
            <li>Order ID</li>
            <li>Payout ID</li>
            <li>Status</li>
            <li>Created At</li>
            <li>Updated At</li>
         </ul>
      </div>
</details>

<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
