> **Note: WPFunnels is available in the free and pro versions.**

1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the **_“Create Integration”_** button.

   <Image src="trigger/_create-integrations" alt="WPFunnels" w="1020" h="350" />

2. Now select the **WPFunnels** that you want to integrate with Bit Integrations **Actions**.

   <Image src="trigger/wp-funnels/select-trigger" alt="WPFunnels" w="1020" h="350" />

3.	Bit integrations will automatically generate a **webhook URL** & you have to copy that link. And click on the **`fetch`** button. Now navigate to WPFunnels -> `click three dots (on the top right side)` -> **Webhooks** -> Add Webhook. Fill up the fields Paste the copied Webhook URL into the URL field and select the Request Method and Format from the drop-down list based on which you want to fetch data and save the Webhook with other information. Submit the form then go to the Bit integration page and click on the **`set action`** button.

   <Video src="wp-funnels-fetch-data" alt="WPFunnels" w="1020" h="350" />
   
> **Note:** Do not copy the webhook URL until the Fetch button is activated.
> **Note: You must install the `WPFunnels pro` plugin to set webhooks in WPFunnels.**

<br/>

:::info
  After completing these steps, the **Actions** list will appear, and select the action you want to integrate with your selected **Trigger**. Now you have to **authorize** the Action. To know more about authorization, Go to **Actions** from the left side menu and select the Action you want to integrate with your chosen trigger.
:::
