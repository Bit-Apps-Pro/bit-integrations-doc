eForm Integration With Telegram

To integrate with Telegram follow, the below steps.

Process of Authorization and Integration

  1. At First, Open your Bit Integrations Dashboard and create an integrations by clicking the “Create Integrations” button.

  1. Now select eForm that you want to integrate with Telegram.

  2. Select the form name from the Drop-Down list that you want to integrate with Telegram. And click on the “Next” button. And select Telegram.

  3. Integration Name: Set the proper integration name Default it's taken “Telegram”.

  4. To get the API Key, visit your telegram account. Now, search for @BotFather. If you have created a bot before, then just copy the API key & past the key on the API Key field on the eForm integration page else, create a new one by sending a command /newbot to @BotFather

  1. Authorize the entire settings.

  2. Click on Next for further integration settings ahead.

  3. Click on the drop-down & select the specific chat list you have created earlier on Telegram.

    A. Sometimes you may not find the lists of your telegram account. One thing you can do that – go to your telegram account, then remove the administrator’s bot from that specific list or group & add again the administrator’s bot.

    B. To remove the bot & re-add the bot follow the path: Click on your desired list/group/Channel => Go to settings of that list/group/Channel => Administrator => Remove the promoted bot & add again the administrator’s bot

  1. Select the parse_mode method to send text notifications to the specified group or channel. You can use either HTML or Markdown format to send a message.

  2. Now, type a message as your preference. You can select Form Fields to fetch the user information if you want. Also, you can mention any group or channels from this message field.

Actions

  1. You can select the action Attachment right below the Message’ field. Enabling this action, you can send – Audio, Video, Photo, or Document from eForm to Telegram. After this, click on the Next option.

Save Integration

  1. To successfully integrate your form with Telegram click the Finish & Save button.

    We have an option in every integration "Timeline," where you can check integration success or error status.
