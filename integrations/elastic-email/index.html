Authorization Process With Elastic Email

To integrate with Elastic Email follow, the below steps.

Process of Authorization and Integration

  1. Search & Select Elastic Email from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken Elastic Email.

  3. After that, To get the API key, click on Elastic Email API Console or Login your Elastic Email dashboard => click on My Account => Setting => Create Additional API Key . You need to grant Full Access permission to make the API successful. Now copy the API key and paste it into the integrations dashboard. And Click on “Authorize” button

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the List from the drop-down option. You have to create a list before on your Elastic Email account.

  3. You will find the Map Field option. You have to map all the necessary form fields with the related Elastic Email field. Here ‘Email‘ field is the default for Elastic Email. Now, click on the Next button to move forward for further settings.

  4. Now save the integration by clicking the Finish & Save Button

  5. We have an option in every integration Timeline where you can check integration success or error status.

  6. Also, you can Edit Clone Delete the integration.
