Authorization Process With Fluent Support

Note: Fluent Support is available in the free and pro versions.

To integrate with Fluent Support follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Fluent Support from the actions pages.

  2. For username & password: For username, put the login user name, and for the password, you have to create Application Passwords under user, to create an application password. Navigate to the user and edit the user, and you will find the create Application Passwords option and create the password. Now copy the password and paste it into integration setting

  3. After successful authorization. You will find the Map Field option. Map all the mandatory Fluent Support fields to the corresponding fields in your form/task.

Actions:​

  1. Support Staff: Ticket Ownr is mandatory in Fluent Support. You have to select the owne. Ticket Owner is mandatory to create a ticket on Fluent support. Select the support staff from drop-down option.

  2. Client Priority: By enabling this option, you can set the priority for every ticket. Select priority from the drop-down option of your choices.

  3. Now save the integration by clicking the Finish & Save Button.

  4. Bit Integrations have an option in every integration Timeline where you can check integration success or error status.

  5. Also, you can Edit Clone Delete the integration.
