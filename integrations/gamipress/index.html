Authorization Process With GamiPress

Note: GamiPress is only available on the pro version.

To integrate with GamiPress follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select GamiPress from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken GamiPress.

  3. Now, click on the connect button to complete authorization settings.

  4. Click on Next to set up the other integration settings ahead.

Actions List​

Award rank to the user​

  1. After the authorization process, select the Action as Award rank to the user. Then select the Rank Type from the drop-down option and the Rank from your choice's drop-down option. If you don't see any Rank Type and Rank. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, The user is awarded a reward rank.

  1. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Award an achievement to the user​

  1. After the authorization process, select the Action as Award an achievement to the user. Then select the Achievement Type from the drop-down option and the Achievement from your choice's drop-down option. If you don't see any Achievement Type and Achievement. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, An achievement will be awarded to the user.

  1. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Award a points to the user​

  1. After the authorization process, select the Action as Award points to the user. Then select the Point Type from the drop-down option of your choice drop-down option. If you don't see any Point Type. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, will increase users' points to Point Type.

  1. You will find the Map Field option. You have to map the necessary form field with the related GamiPress field. Here Point field is the default for GamiPress. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Revoke a rank form the user​

  1. After the authorization process, select the Action as Revoke a rank form the user. Then select the Rank Type from the drop-down option and the Rank from your choice's drop-down option. If you don't see any Rank Type and Rank. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, Revoke a rank from the user.

  1. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Revoke an achievement form the user​

  1. After the authorization process, select the Action as Revoke an achievement from the user. Then select the Achievement Type from the drop-down option and the Achievement from your choice's drop-down option. If you don't see any Achievement Type and Achievement. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, Revoke an achievement form the user.

  1. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Revoke points from the user​

  1. After the authorization process, select the Action as Revoke points from the user. Then select the Point Type from the drop-down option of your choice drop-down option. If you don't see any Point Type. Click on the refresh button.
  • If you select this option from the drop-down. When users submit/complete the form/task, will decrease users' points to Point Type.

  1. You will find the Map Field option. You have to map the necessary form field with the related GamiPress field. Here Point field is the default for GamiPress. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

Timeline​

  1. We have an option in every integration Timeline where you can check integration success or error status.

  2. Also, you can Edit Clone Delete the integration.
