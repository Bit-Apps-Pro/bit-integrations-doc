Authorization Process With Getgist

To integrate with Getgist follow, the below steps.

Process of Authorization and Integration

  1. Search & Select Getgist from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'GetGist'.

  3. After that, you have to copy the API key from your Getgist account & paste the key on the field. To get the API key, click on Getgist API Console or go to your Getgist dashboard =>click on Account Setting => Setting =>API Key. Now copy the API key and paste it into the integrations dashboard. And Click on “Authorize” button.

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the User Type from the drop-down option User or Lead.

  3. You will find the Map Field option. You have to map all the necessary fields with the related Getgist field. By default, “Email” field of Getgist will be available. You can also send custom values by mapping your required field. then, click on the Next button to move forward for further settings.

  4. Now save the integration by clicking the Finish & Save Button

  5. We have an option in every integration Timeline where you can check integration success or error status.

  6. Also, you can Edit Clone Delete the integration.
