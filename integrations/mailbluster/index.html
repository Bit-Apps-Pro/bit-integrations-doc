Authorization Process With MailBluster

Note: MailBluster is only available on the pro version.

To integrate with MailBluster follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select MailBluster from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'MailBluster'.

  3. After that, you have to copy the API key from your MailBluster account & paste the API key on the field.
    To get MailBluster API key

  • First login your MailBluster account
  • Select your preferred brand
  • Go to setting > API Keys
  • Click on Create API keys
  • Copy the API Key and paste into API Key field of your authorization form. And Click on Authorize button

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the Type of your choices from the drop-down options.

  3. You will find the Map Field option. You have to map all the necessary fields with the related MailBluster field. By default, Email field of MailBluster will be available. You can also send custom values by mapping your required field. then, click on the Next button to move forward for further settings.

  4. Now save the integration by clicking the Finish & Save Button

Actions​

  • Add Tags: You can enable the action Add Tags Field to add the tags of the contact. In addition, you can add multiple tags from the Drop-Down list.

  • Update Lead: You can also update the submitted data by enabling the action called Update MailBluster. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address.

  • Double Op-tin: By enabling the Double-Opt-in action admin can ask permission from users if they want to subscribe to MailBluster or not.

  1. We have an option in every integration Timeline where you can check integration success or error status.

  2. Also, you can Edit Clone Delete the integration.
