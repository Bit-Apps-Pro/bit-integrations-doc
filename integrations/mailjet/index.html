Authorization Process With Mailjet

Note: Mailjet is only available on the pro version.

To integrate with Mailjet follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Mailjet from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'Mailjet'.

  3. After that, you have to copy the API key from your Mailjet account & paste the API key on the field.
    To get Mailjet client id and client secret

  • First login your Mailjet account

  • Go to setting > advanced setting > developer's corner > API Keys

  • Create a NEW API Key or copy the client id and client secret from the previously created API key

  • Then paste into client id and client secret field of your authorization form. And Click on Authorize button

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the List of your choices from the drop-down options, and you can select multiple lists. If you don't see any lists. Click on the refresh button.

  3. You will find the Map Field option. You have to map all the necessary fields with the related Mailjet field. By default, Email field of Mailjet will be available. You can also send custom values by mapping your required field and custom field. If you don't see any fields and missing. Click on the refresh button.

  4. Now save the integration by clicking the Finish & Save Button

Actions​

  • Is excluded from campaigns: Indicates whether the contact is added to the exclusion list for campaigns or not. An excluded contact will not be receiving any marketing emails.

  1. We have an option in every integration Timeline where you can check integration success or error status.

  2. Also, you can Edit Clone Delete the integration.
