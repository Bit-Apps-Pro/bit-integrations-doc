PipeDrive with WordPress Forms, WooCommerce, LMS, Membership, and Other Plugins Integrations.

Note: PipeDrive is available in the free and pro versions.

Pipedrive integrations via Bit Integrations is a seamless way to connect your PipeDrive account with other business tools and applications. With a wide range of integrations available. You can send data from your WordPress forms, Page/Theme Builder, WooCommerce, LMS, Membership, and other plugins to your PipeDrive account. For more details, refer to the integration guide provided.

Process of Authorization and Integration​

  1. Search & Select PipeDrive from the actions pages

  2. API Token: To get API Token

  • Open your PipeDrive account.

  • Click on the Profile icon then Navigate to Personal preferences

  • Navigate to API

  1. After that, you have to copy API Token from your PipeDrive account & paste them into the integration setting. And Click on the Authorize button. After successfully authorized done. Click on Next to set up the other integration settings ahead.

  2. Module: Choose the desired module from the options in the drop-down list.

  3. Organization: Select the organization under the respective module. When you select an organization under the respective module in Pipedrive CRM, you will be able to view all the people related to that organization in the selected Organization view.

  4. People: Select the people from the drop-down list which you have already created in your Pipedrive CRM.

  1. You will find the Map Field option. You have to map all the necessary fields with the related Pipedrive field. After mapping the desired fields, click on the Next button.
    ✅ Available Pipedrive mapping fields
    • Title
    • Value
    • Expected close date

Related list​

You can send data to the Pipedrive-related list from Trigger. Select the required related list module and layout from the drop-down provided. Then map the mandatory field. To map more fields, click on the (+).

Actions​

  1. Owners: This option allows you can set a record owner of the sent record.

  2. Labels: This option allows you to add the label of the record to your Pipedrive.

  3. Currency: This option allows you to set the currency to the sent record

  4. Visible To: This option allows you to control the visibility of the record that will be sent to the integration.

  5. Now save the integration by clicking the Finish & Save Button

  6. Bit Integrations is an option in every integration called Timeline where you can check integration success or error status.

  7. Also, you can Edit Clone Delete the integration.
