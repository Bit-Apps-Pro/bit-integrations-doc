Authorization Process With Post Creation

Note: Post Creation is only available on the pro version.

To integrate with Post Creation follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Post Creation from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'Post Creation'.

  3. After navigate to the integration page, select the Post Type you have created for the integration. For example, we have selected Photo Album as custom post type.

  4. Now, select Post Status, Author & Comment Status according to the requirement.

  5. For the field mapping option, map the form fields with the general/default post fields.

  6. Click on Next to set up the other integration settings ahead.

  7. For the field mapping option, you need to configure two types of mapping of two popular plugins (ACF, Meta Box) as needed -

    • Then map the form fields with the related ACF or Meta Box fields.
    • Lastly map the file upload field.

  8. Click on the Next button to move forward for further settings.

  9. Now save the integration by clicking the Finish & Save Button.

  10. We have an option in every integration Timeline where you can check integration success or error status.

  11. Also, you can Edit Clone Delete the integration.
