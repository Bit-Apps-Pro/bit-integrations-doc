Authorization Process With Sendinblue

Note: Sendinblue is only available on the pro version.

To integrate with Sendinblue follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Sendinblue from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken “Sendinblue”.

  3. API Key: You have to copy the key from the Sendinblue dashboard & paste the key on the field. If you don’t have any create a new one. To get a new API key, go to Sendinblue API CONSOL=> SMTP & API=> Create a new API key. Register all the information and get your API key.

  4. Now click Authorize & you are ready to integrate with your Sendinblue account.

  5. Select the list from the drop-down option. You have to create a list on the Sendinblue dashboard & that will show on the drop-down list. Also, you can select multiple lists.

  6. When you fill-up the contact list field, you will find the Map Field option. Map all the mandatory CRM fields to the corresponding fields in your form/task.

Actions​

  1. You can also update the submitted data by enabling the action called Update Sendinblue. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled then the new data will be updated regarding that email address.

  2. Now save the integration by clicking the "Finish & Save." Button

  3. We have an option in every integration "Timeline" where you can check integration success or error status.
