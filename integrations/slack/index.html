Authorization Process With Slack

To integrate with Slack follow, the below steps.

Process of Authorization and Integration

  1. Search & Select Slack from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'Slack'.

  3. After that, you have to copy the API key from your Slack account & paste the key on the field. To get the API key, click on Slack API Console or go to api.slack.com/apps
    Setup Proccess

    1. click on Create New App/Create an App: From scratch.
    2. Type your app name
    3. Select your workspace from the Drop-Down list that you want to send the message. and click on Create App button.
    4. In the left sidebar menu, navigate to OAuth & Permissions tab under Features.
    5. Scroll Down and go to User Token Scopes under scopes section and add scopes chat:write, channels:read, channels:write, files:read, files:write.
    6. Go to OAuth Tokens for Your Workspace section and click on install to workspace button.

  4. Now copy the API key and paste it into the integrations dashboard. And Click on ‚ÄúAuthorize‚Äù button.
