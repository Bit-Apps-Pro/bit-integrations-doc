Authorization Process With SliceWp

Note: SliceWp is available in the free and pro versions.

To integrate with SliceWp follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select SliceWp from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken SliceWp.

  3. Now, click on the connect button to complete authorization settings.

  4. Click on Next to set up the other integration settings ahead.

Actions List​

Add commission to user's affiliate​

If you select this option from the drop-down. When users submit/complete the form/task, Add commission to the user's affiliate.

  1. After the authorization process, select the Action as Add commission to user's affiliate.

  2. You will find the Map Field option. You have to map all the necessary fields with the related SliceWp field. By default, Commission Date field of SliceWp will be available. You can also send custom values by mapping your required field. Now, click on the Next button to move forward for further settings.

  3. Now save the integration by clicking the Finish & Save Button

Actions​

Add Status: Status is mandatory in SliceWP. You can add status (Pending, Paid, Unpaid, Rejected) from the Add Status.

Add Type: You can add a type (Sale, Subscription) from the Add Type.

  1. We have an option in every integration Timeline where you can check integration success or error status.

  2. Also, you can Edit Clone Delete the integration.
