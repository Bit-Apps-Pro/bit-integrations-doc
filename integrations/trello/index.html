Authorization Process With Trello

To integrate with Trello follow, the below steps.

Process of Authorization and Integration

  1. Search & Select Trello from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken 'Trello'.

  3. After that, to get the API key, click on Trello API Console
    Setup Proccess

    1. Copy Authorized Redirect URIs
    2. Scroll down and go to 'Allowed Origins' section, paste ‘New Allowed Origin’ field.
    3. Click on Submit button
    1. Now copy the API key and paste it into the integration’s dashboard. And Click on “Authorize” button.

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, Select the Boards from the drop-down option. You have to create a Boards before on your trello account. And select the list form the drop-down option.

  3. You will find the Map Field option. You have to map all the necessary form fields with the related Trello Board field. Here ‘Name ‘field is the default for Trello Board.

Actions

  1. If you enable this option. You can display your message at the top or bottom of the list as you wish. Now, click on the Next button to move forward for further settings.

  2. Now save the integration by clicking the Finish & Save Button

  3. We have an option in every integration Timeline where you can check integration success or error status.

  4. Also, you can Edit Clone Delete the integration.
