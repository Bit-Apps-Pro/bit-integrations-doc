Authorization Process With Vbout

Note: Vbout is available in the free and pro versions.

To integrate with Vbout follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Vbout from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken Vbout.

  3. After that, you have to copy the API Token from your Vbout account & paste the key on the field.
    To get the API Token,

  • click on Vbout API Token or go to your Vbout dashboard

  • click on Profile icon => Setting => Navigate to API Integrations

  • Now copy the Your Unique API User Key and paste it into the integration’s dashboard. And Click on “Authorize” button

  1. Click on Next to set up the other integration settings ahead.

  2. After completing the authorization process, select the Contacts list that you have created before on Vbout. By default, the name of the list is Sample list. Then elect the Contact Status from the drop-down option (Active, Unconfirmed). If you don't see any Contacts list. Click on the refresh button.

  3. You will find the Map Field option. You have to map all the necessary fields with the related Vbout field. By default, “Email” field of Vbout will be available. You can also send custom values by mapping your required field. Now, click on the Next button to move forward for further settings. If you don't see any Vbout fields. Click on the refresh button.

  4. Now save the integration by clicking the Finish & Save Button

  5. We have an option in every integration "Timeline" where you can check integration success or error status.

  6. Also, you can Edit, Clone, and, Delete the integration.

Actions​

  • Update Contact: If the user already exists on your Vbout account, It will update the contacts with other field information based on the email address.
