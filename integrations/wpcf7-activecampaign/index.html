Authorization Process With ActiveCampaign

Note: Active Campaign is only available on the pro version.

To integrate with ActiveCampaign follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Active Campaign from the actions pages

  2. Integration Name: Set the proper integration name. Default it's taken “ActiveCampaign”.

  3. Access API URL & API key: To get the URL & key : log in to ActiveCampaign dashboard=> Settings=> Developer=> Get your API URL & Key.You have to copy the API URL & Key from the Active Campaign Developer section & paste the URL & key on the fields of integration settings.

  4. Now click Authorize, & you are ready to integrate with your ActiveCampaign account.

  5. After successful authorization. Select the List from the drop-down option. You have to create a list before on your ActiveCampaign account. You can add a Tag [ also multiple tags] for the Contacts that you have created before on your ActiveCampaign account

  6. You will find the Map Field option. You have to map all the necessary form fields with the related ActiveCampaign field. Here ‘Email‘ field is the default for ActiveCampaign.

Actions​

  1. You can also update the submitted data by enabling the action called Update ActiveCampaign. Suppose the user already submitted data under an email address. Again they want to submit data under the same email, if the action is enabled, then the new data will be updated regarding that email address.

  2. Now save the integration by clicking the "Finish & Save." Button

  3. We have an option in every integration "Timeline," where you can check integration success or error status.
