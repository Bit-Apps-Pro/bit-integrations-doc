Authorization Process With Zoho Marketing Hub

Note: Zoho Marketing Hub is only available on the pro version.

To integrate with Zoho Marketing Hub follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Zoho Marketing Hub from the actions pages

  2. Integration Name: Set the proper integration name Default it's taken “Zoho Marketing Hub”.

  3. Select a data center from the dropdown. Then click on the Zoho API Console. Navigate to API Console-> Get Started-> Server-based Applications.

  1. Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click CREATE button

  1. A Client ID and a Client Secret will be generated. Then go to the Client Secret tab

  2. Then copy the Client ID and Client Secret and paste it to the Bit Integrations integration page. Then click on the Authorize button (wait for a few seconds). After the successful authorization click on the Next button

  3. Select the list for the lead collection from the drop-down. Before that make sure you have already created a list on your marketing hub account.

  1. When you fill-up the contact list field, you will find the Map Field option. You have to map all the necessary form fields with the related Zoho Marketing Hub.

  1. After clicking on Finish & Save, the Mailchimp integration is ready.

  2. We have an option in every integration "Timeline," where you can check integration success or error status.
