Authorization Process With Zoho Recruit

Note: Recruit is only available on the pro version.

To integrate with Zoho Recruit follow, the below steps.

Process of Authorization and Integration​

  1. Search & Select Zoho Recruit from the actions pages

  2. Integration Name: Set the proper integration name Default it's taken “Zoho Recruit”.

  3. Select a data center from the dropdown. Then click on the Zoho API Console. Navigate to API Console-> Get Started-> Server-based Applications.

  1. Fill those fields (Homepage URL and Authorized Redirect URLs) the same as your Integration Homepage. Then click CREATE button

  2. A Client ID and a Client Secret will be generated. Then go to the Client Secret tab

  3. Then copy the Client ID and Client Secret and paste it to the Bit Integrations integration page. Then click on the Authorize button (wait for a few seconds). After the successful authorization click on the Next button

  4. Select a Module from the drop-down list. Map all the mandatory Recruit fields to the corresponding fields in your form.

  5. If you use File Upload field should to map this separately.

Related List​

  1. You can send data to Zoho Recruit related list from Trigger. Select the required related list module and layout from the drop-down provided. Then map the mandatory field. To map more fields, click on the (+).

Actions​

  1. Workflow: Select this option to trigger the Zoho Recruit workflow for the selected module. If you didn’t select this option Zoho Recruit workflow for the selected module doesn’t work.

  2. Approval: Select this option to automatically send form data to your Zoho Recruit approval list instead of adding them.

  3. Upsert Record: : When you choose the Upsert Record option, if a record with the identical value exists in Zoho Recruit, then it will be updated with the new values. If not, a new record will be created.

  4. Add Note: Select this action to add a note from Trigger to Zoho Recruit account automatically.

  5. Record Owner: YSelect this option you can set a record owner of the sent record. For this, you have to copy the Owner ID from Zoho Recruit and paste it here.

Save Integration​

  1. After giving all the required information, click on Finish & Save. Now your created Zoho Recruit API is ready for implementation
    We have an option in every integration "Timeline," where you can check integration success or error status.
