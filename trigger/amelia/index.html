Select Amelia As A trigger

Note: Amelia is available on the free and pro version.

  1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the “Create Integration” button.

  2. Now select the Amelia that you want to integrate with Bit Integrations Actions.

  3. Now copy the Webhook URL and click on the Fetch button. Now navigate to Setting -> Integration -> Webhook -> Add New Webhook from your Amelia account. Paste the copied Webhook URL into the URL field and select the Type and Action from the drop-down list based on which you want to fetch data and save the Webhook with other information.

    After completing these steps create an event or appointment with the status of the action. Or please follow the below video here is a demo video to Fetch Amelia's data

  1. After successfully Fetched the data then click on Set Action button
info

After completing these steps, the Actions list will appear, and select the action you want to integrate with your selected Trigger. Now you have to authorize the Action. To know more about authorization, Go to Actions from the left side menu and select the Action you want to integrate with your chosen trigger.
