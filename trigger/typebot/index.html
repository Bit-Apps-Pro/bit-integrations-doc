Select Typebot As A trigger

Note: Typebot is available on the free and pro version.

  1. At First, Open your Bit Integration Dashboard and create an integrations by clicking the “Create Integration” button.

  2. Now select the Typebot that you want to integrate with Bit Integrations Actions.

  3. Bit integrations will automatically generate a webhook URL & you have to copy that link. And click on the fetch button. Now navigate to typebot dashboard -> select your typebot -> Add integration(webhook) -> click on configure . Paste the copied Webhook URL into the URL field. Click on to Test the request button. Then go to the Bit integration page and click on the set action button.


info

After completing these steps, the Actions list will appear, and select the action you want to integrate with your selected Trigger. Now you have to authorize the Action. To know more about authorization, Go to Actions from the left side menu and select the Action you want to integrate with your chosen trigger.
